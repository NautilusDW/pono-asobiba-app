# ナレーション 漢字交じり版スクリプト (AI Studio TTS 入力用)

ゲーム画面上は全てひらがな表示だが、**AI Studio の Gemini TTS に入力する際は、易しい漢字を交ぜた方がアクセント・イントネーションが正しく生成される**。このドキュメントは TTS 収録用のコピペ原稿。

## 運用ルール

- **ゲーム内テキスト**: ひらがなのまま（[writing/index.html](../writing/index.html) の `OPENING_SCENES` / `ENDING_SCENES` `text:` フィールド）
- **TTS 入力**: このドキュメントの漢字交じり版を AI Studio にコピペして生成 → 出力 mp3 を `assets/audio/storyboard/voice/` に配置
- **画面とのズレ**: 音声が読み上げる内容と画面文字は同一（漢字版は音声のためだけのマッチング）

## AI Studio 設定（ナレーター）

- **Speaker**: `Iapetus`
- **Scene**: ファンタジーRPG の語り部が、王国で起きている出来事を子供に向けて話して聞かせる場面。
- **Sample Context**: 5歳児向けゲームのナレーション。王国の物語を子供にはっきりと伝える場面。通常の声量で、普段の会話の明瞭さで、朗々と普通に読む。ささやき声は使わない。

## AI Studio 設定（王様）

- **Speaker**: `Charon`
- **Scene**: ファンタジーRPG の国王。魔王に姫を奪われ国を呪われた状況で、勇者に必死の思いで救出を命じる場面。
- **Sample Context**: 威厳のある王の声。低く太いが、愛する娘と国を失いかけている悲痛さと、勇者への最後の希望を込めた強い呼びかけ。通常の声量で、はっきりと朗々と発声する。ささやき声は使わない。

## 正規化（必須）

AI Studio 出力は 48kHz stereo 192kbps で既存 mp3 より -5〜-7dB 低い傾向。**必ず -14 LUFS に揃える**:

```bash
ffmpeg -y -i INPUT.mp3 -af "loudnorm=I=-14:LRA=11:TP=-1.5" -c:a libmp3lame -b:a 128k OUTPUT.mp3
```
