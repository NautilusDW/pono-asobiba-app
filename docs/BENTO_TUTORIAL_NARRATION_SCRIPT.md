# おべんとうチュートリアル 現行ナレーション原稿

> **ステータス: 原稿のみ。音声は未生成・未配置。**
> 2026-07-12 時点の `bento/index.html` の現行シナリオに合わせた収録用正本です。
> `BENTO_TUTORIAL_MOCK_VOICE` は現在 `true` で、`tut2_*.mp3` はまだありません。

## 1. 読み上げ方

- 女性の第三者ナレーターとして、やさしく、少しゆっくり読む。
- ポノや動物の声にはせず、「わたし」「ぼく」などのキャラクター一人称は使わない。
- 子どもが聞く文は、ひらがな・カタカナで読む。「タップ」は使わず、すべて「タッチ」にする。
- 一つの音声では、基本的に一つの操作だけを案内する。操作が終わるまで次の音声を重ねない。
- 収録時は Gemini Leda を 1.15 倍速で使い、生成後に faster-whisper で原稿と照合する。

### メイン2個目の案内

1個目によって2個目の指定が変わるため、ここは録音せず画面表示だけにします。

- 1個目がハンバーグ以外: `もう ひとつ ハンバーグを えらぼう！`
- 1個目がハンバーグ: `もう ひとつ おかずを えらぼう！`（メインなら自由に選べる）

## 5. 音声にしない動的な案内

次の文は状態や残り回数で変わるため、画面の吹き出しとトーストだけにします。

- `あと 3こ` など、小さいおかずの残り数
- 仕切りの残り数
- メイン2個目の品名
- `それは ちがうよ` から始まる誤操作案内
- 目・鼻・口を置く途中の `もう ひとつ` や `さいごに` の短い案内
- `できたね`、`ぜんぶ そろったね` などの途中確認
- レタスを別の場所へ置こうとしたときの、対象名入り再案内

`tut2-leaf-target` はすぐに `tut2-leaf-pick` へ進む内部ステップなので、専用音声は作りません。
