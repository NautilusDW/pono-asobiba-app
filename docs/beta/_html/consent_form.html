このおためし版について

お子さんと一緒に web で遊べる知育アプリ 「ポノのあそびば」 のおためし版です。 ご家庭で2週間ほど遊んでいただき (保育園は内容を見ません)、 簡単なアンケートにお答えいただけたら助かります。 ご負担は全部あわせて 30分以内 を目安にしています。

この情報は、 保育園には一切共有されません。

3つのお約束

  1. お子さんを特定できる情報 (お名前・顔写真・声・位置情報) は、 一切収集しません。
  2. あそびのきろく(どのゲームでどれくらい遊んだか)は、名前と結びつかない符号を使って自動的に記録されます。 ボタンを押して送る項目(保護者の任意メモなど)のほかにも、自動で送られる記録があります。
  3. いつでも、 理由なくやめられます (遊ぶのをやめるだけで、 ご連絡不要)。 遊ぶのをやめていただくだけで OK です。
  4. このチラシ・QR コード・URL は、 保育園以外の方には共有しないでください。
