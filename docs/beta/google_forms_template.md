# ポノのあそびば βテスト Google Forms コピペシート

> このファイルは Google Forms に**そのままコピペして 2本のアンケート (中間 / 最終) を作る**ためのテンプレート集です。
> 並走する手順書: [`google_forms_setup_guide.md`](./google_forms_setup_guide.md) を先に読んでから戻ってきてください。

---

## 設問の意図 (作る前に 30秒で目を通す)

### 中間アンケート (Week 1 末・5問・90秒)
- **Q1**: 子供から「もう一回やりたい」と自発的にリクエストがあったか → **リテンション先行指標**。 数字より行動の有無で測る。
- **Q2**: 操作で迷った画面 → **UX 詰まりポイント**の早期発見。 自由記述ではなく選択式にして集計を軽くする。
- **Q3**: Q2 の詳細を任意で書ける長文欄 → 選択式では拾えない一次情報を、 書きたい人だけが書ける形で補完。
- **Q4**: クラッシュ/フリーズ/真っ白の有無 → **安定性 P0 バグ**を 1週間で炙り出す。
- **Q5**: Q4 の詳細を任意で書ける長文欄 → どのゲーム/どの操作かを書きたい人だけ書く再現情報の窓口。

### 最終アンケート (Week 3 頭・13問・7-8分)
- **子の体験 (Q1-Q4)**: 楽しさ・繰り返し度・お気に入りゲーム・年齢適合性 → アプリそのものの評価軸
- **親評価 (Q5-Q9)**: 安心して渡せたか・操作迷い (Q7 で詳細任意記述)・広告/課金圧の不快感・親が知育として価値を感じたか
- **推奨意向 (Q10-Q12)**: 5段階満足度 / NPS / 創設メンバー継続意向 → go/no-go 判定の数値根拠
- **自由記述 (Q13)**: 言葉にならない引っかかりと、 次の v1.1 で欲しい機能

# 【アンケート 1】 中間アンケート

## フォーム基本設定

- **タイトル**: `ポノのあそびば おためし版 中間アンケート`
- **説明文** (タイトル直下にコピペ):

```
ポノのあそびば のβテストに ご参加いただき ありがとうございます。
1週間 つかってみての ざっくりした感想を 5問だけ おしえてください。
所要時間は 約90秒です。

回答は匿名で集計され、 個人を特定できる情報 (お名前/お顔/お声/位置情報) は一切収集しません。
ご回答いただいたデータは 2026年12月31日までに 全て削除します。
```

## Q1 — 自発リクエスト

- **質問タイトル**: `この1週間で お子さまから 「もう一回 ポノのあそびばで あそびたい」と 自発的に リクエストがありましたか？`
- **回答形式**: **ラジオボタン**
- **必須**: はい

```

2-3回あった
1回だけあった
リクエストはなかった
わからない / 覚えていない
```

```

ゲームクリア後 (次にどこへ行くか分からない)
スタンプ帳 / シール画面
その他 (最終アンケートで詳しく教えてください)
```

## Q2 — 一番くりかえし遊んだゲーム

- **質問タイトル**: `この2週間で お子さまが 一番くりかえし遊んだゲームは どれですか？`
- **回答形式**: **ラジオボタン**
- **必須**: はい
- **ヘルプテキスト**:
  ```
  「だいたいこれが多かった」 という感覚で 1つ選んでください。
  ```
- **選択肢** (各選択肢に**ゲームアイコン画像を添付**する):
```
ポノとつくろう いろどりべんとう
ポノのなかよしパズル
ポノとランタンのめいろ
フクロウはかせのなぞなぞ
ポノのおとタッチ
シールちょう
どれもだいたい同じくらい
あまり遊ばなかった
```

- **画像添付指示** (各選択肢ごと):
  - ポノとつくろう いろどりべんとう: `assets/ui/title/bento.png`
  - ポノのなかよしパズル: `assets/ui/title/puzzle.png`
  - ポノとランタンのめいろ: `assets/ui/title/maze.png`
  - フクロウはかせのなぞなぞ: `assets/ui/title/quizland.png`
  - ポノのおとタッチ: `assets/ui/title/oto.png`
  - シールちょう: (画像不要、 テキストのみ。 もし `assets/ui/title/sticker.png` 等の適切な画像があれば後付け可)
  - **添付手順**: Forms 編集画面で 選択肢の右端にマウスを当てると現れる **画像アイコン** → アップロード → 上記パスの画像を指定。
