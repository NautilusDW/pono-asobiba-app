# ポノのあそびば βテスト Google Forms コピペシート

> このファイルは Google Forms に**そのままコピペして 2本のアンケート (中間 / 最終) を作る**ためのテンプレート集です。
> 並走する手順書: [`google_forms_setup_guide.md`](./google_forms_setup_guide.md) を先に読んでから戻ってきてください。

---

## 設問の意図 (作る前に 30秒で目を通す)

### 中間アンケート (Week 1 末・5問・90秒)
- **Q1**: 子供から「もう一回やりたい」と自発的にリクエストがあったか → **リテンション先行指標**。 数字より行動の有無で測る。
- **Q2**: 操作で迷った画面 → **UX 詰まりポイント**の早期発見。 自由記述ではなく選択式にして集計を軽くする。
- **Q3**: Q2 の詳細を任意で書ける長文欄 → 選択式では拾えない一次情報を、 書きたい人だけが書ける形で補完。
- **Q4**: クラッシュ/フリーズ/真っ白の有無 → **安定性 P0 バグ**を 1週間で炙り出す。
- **Q5**: Q4 の詳細を任意で書ける長文欄 → どのゲーム/どの操作かを書きたい人だけ書く再現情報の窓口。

### 最終アンケート (Week 3 頭・16問・8-10分)
- **子の体験 (Q1-Q4)**: 楽しさ・繰り返し度・お気に入りゲーム・年齢適合性 → アプリそのものの評価軸
- **親評価 (Q5-Q9)**: 安心して渡せたか・操作迷い (Q7 で詳細任意記述)・広告/課金圧の不快感・親が知育として価値を感じたか
- **推奨意向 (Q10-Q15)**:
  - **Q10**: 5段階総合満足度 → ざっくり感情温度
  - **Q11**: NPS (0-10、 推奨意向) → 「他人に勧めたい度」 の数値根拠 (PMF とは別軸)
  - **Q12**: **Sean Ellis PMF** (もし使えなくなったらどう感じるか、 4択) → 「とても残念」 40%以上で product-market fit 達成シグナル。 NPS とは独立指標
  - **Q13**: **紹介時の言葉 (Word-of-Mouth voice)** 自由記述 → LP/広告コピーの一次素材として、 ユーザーが実際に使う言葉を回収 ([[feedback_lp_marketing_copy_rewrite]] 準拠)
  - **Q14**: **新規購入意向 (Early Access ¥980)** ラジオ → β特典抜きで、 一般ユーザー視点で Early Access 期間限定セール価格 ¥980 (後に正式版 ¥1,480 へ値上げ予定) を払う意思があるか。 Q15 (convert-from-free) とは別軸
  - **Q15**: 創設メンバー継続意向 (無料特典がなかったら払うか) → 既に使った人の retention 価格弾力性
- **自由記述 (Q16)**: 言葉にならない引っかかりと、 次の v1.1 で欲しい機能

> **注意**: Q11 (NPS) は 「他人に勧める意向」 を測るが、 product-market-fit (使えなくなった時の喪失感) を直接測るのは Q12 (Sean Ellis PMF) のみ。 NPS が高くても PMF が低いケース (悪くないけど無くても困らない) は launch 判断を見送る根拠になる。 Q14/Q15 は購入意向を 「新規視点」 と 「既存ユーザー視点」 の二軸で分けて測る。
