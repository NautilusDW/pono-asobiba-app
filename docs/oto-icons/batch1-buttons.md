# 音タッチ ボタン素材シート Batch 1 — Codex 発注仕様書

## ゴール
子供向け音遊びアプリ「音タッチ」のボタン 23 個（楽器 8 + 音色 6 + 速度 3 + UI 制御 6）を、1 枚の透過 PNG アセットシートで一気に発注する。**ポップでカラフルなゲーム UI**（絵本テイスト・木製・布素材は一切禁止）。アイコンとボタン土台は一体デザインで描き、後で切り出して `assets/images/oto/buttons/` に配置する。

## スタイル方針

### DO
- ポップ、明るい、彩度高め、ゲーム UI 寄り
- 光沢・ハイライト・ソフトグラデ・薄い内側グロー OK
- 円形・ピル型の土台、フラット〜セミフラット、ドロップシャドウは各ボタン内に内包
- 子供が「押せる」と直感する立体感（軽い 3D ボタン風）

### DON'T
- 絵本水彩、木目、布、ステッチ、刺繍、フェルト、紙、ヴィンテージ調
- くすみ色・アースカラー
- ポノ（クマ）を各ボタンに描き込む（タイトルロゴ専用）
- 楽器絵文字をそのまま絵にする（🌸→花、🦋→蝶 等は **NG**。音符＋色で抽象化する）

## アセットシート仕様
- サイズ: **2048 × 2048 px** PNG, 背景透過 (#00000000)
- グリッド: **6 列 × 4 行 = 24 セル**（1 セル空き）。1 セル ≒ 340 × 340 px、内側 20 px パディング、実描画は **256〜280 px 直径目安**でセル中央
- ボタン本体に内包ドロップシャドウ。セル外への影出しは禁止（切り出しのため）
