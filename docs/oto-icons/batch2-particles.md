# 音タッチ Batch 2 — フラットパーティクル/装飾アセットシート 発注仕様

## ゴール
canvas で `fillText` していた絵文字パーティクル ✨⭐💖🎵🎶💫🌟 と、チュートリアル吹き出し冒頭/末尾の装飾 🎵 ✨ を、**透過 PNG のフラットアイコン**に置き換える。シート 1 枚 (3×3 グリッド) で 9 セル分を一括納品。

## スタイル方針

**DO**
- ポップで明るい彩度 (蛍光寄り OK / ネオン基調)
- シンプルなシルエット、子供向けゲーム UI 文法
- 小サイズ表示でも形が読める太めの形状
- 輪郭は **極細の白アウトライン** か **外周ソフトグロー (発光)** のどちらか統一
- パーティクル前提なのでコントラスト強め、彩度高め

**DON'T**
- 絵本水彩・クレヨン質感
- 木目・布・紙などのアナログ素材感 (音色を想起させてしまうため厳禁)
- ポノ (クマ) のモチーフを混ぜない (タイトルロゴ側で担保済み)
- ドロップシャドウ・自前の影 (canvas 側で screen/lighten 合成するため乗算事故源)
- リアル質感、過剰な細部描写

## アセットシート仕様
- サイズ: **1024×1024 PNG / 完全透過 (#00000000)**
- グリッド: **3 列 × 3 行 = 9 セル / 各セル 256×256**
- 各セル: 中央配置、パディング 32px、実描画 192〜220px 目安
- 回転は **0° 基準** (DOM/canvas 側でランダム回転させる)
- 影なし、シート全体に余計な背景・ガイド線なし

## 命名・保存先
- 切り出し後の個別 PNG: `assets/images/oto/icons/oto/<filename>.png`
- 例: `assets/images/oto/icons/oto/spark_sparkle.png`
- シート原本も同ディレクトリに `batch2_sheet.png` で保管

## Codex への余白
Batch 1 (楽器ボタン) と**色温度・彩度の系統**は揃えること。ただしパーティクル用途のためコントラスト・彩度はより強く振って良い。背景に置かれず単体で飛ぶ前提なので、暗背景・明背景どちらでも視認できる輪郭処理を優先。
