ユーザー側で VOICEVOX の女性話者を一通り試聴し、**「雨晴はう」を第一候補**として確定済み（看護師さん風の温かい優しさが、くるみちゃんのお姉さんキャラと最も合うと判断）。

#### ⭐ 第一候補（まずこれで Q01_q.wav を試作）

## 出力仕様

- 形式: wav (48kHz / 16bit / モノラル)
- 命名規則: `q{NNN}_{q|a|b|c|d}.wav` (3桁ゼロパディング)
- Q160 補足版: `q160_a_alt.wav` `q160_b_alt.wav`
- 出力ディレクトリ: `audio/quizland/test/` （納品後にユーザーが配置）

## 読み方ルール（テスト対象の5種）

### ルール1: order_color の英→日変換

画面に英語キー (red/blue/yellow/green/pink/orange/purple/cyan) で定義されているが、実画面では日本語表示される。VOICEVOX も日本語で読み上げる。

| 英語キー | speech (VOICEVOX 入力) |
|---|---|
| red | あか |
| blue | あお |
| yellow | きいろ |
| green | みどり |
| pink | ピンク |
| orange | オレンジ |
| purple | むらさき |
| cyan | みずいろ |

### ルール2: count_total の和語数詞

問題文「○○は いくつ？」に対する答えなので、選択肢は和語数詞で読む。

| 数 | 画面 | speech |
|---|---|---|
| 1 | 1つ | ひとつ |
| 2 | 2つ | ふたつ |
| 3 | 3つ | みっつ |
| 4 | 4つ | よっつ |
| 5 | 5つ | いつつ |
| 6 | 6つ | むっつ |
| 7 | 7つ | ななつ |
| 8 | 8つ | やっつ |
| 9 | 9つ | ここのつ |
| 10 | 10こ | じゅっこ |

### ルール3: opposite のカギ括弧除去

画面表示はカギ括弧「」で囲んでいるが、speech では除去する。

- 例: 画面「「おおきい」の はんたいは？」 → speech「おおきい の はんたいは？」

### ルール4: 同音異義の補足版（Q160）

- Q160 選択肢A「はい」(肺) と選択肢B「い」(胃) は同音異義リスクあり
- 通常版 (`q160_a.wav` = 「はい」、`q160_b.wav` = 「い」) と
  補足版 (`q160_a_alt.wav` = 「ぞうきの はい」、`q160_b_alt.wav` = 「いぶくろの い」) を **両方生成**
- 試聴後にユーザーがどちらを採用するか判断

### ルール5: number_sequence の音読み

3歳児向け新カテゴリ。文脈が「次の数」なので音読みで読む。

| 数 | speech |
|---|---|
| 1 | いち |
| 2 | に |
| 3 | さん |
| 4 | よん |
| 5 | ご |
| 6 | ろく |
| 7 | なな |
| 8 | はち |
| 9 | きゅう |
| 10 | じゅう |

問題文中の数字も音読みに変換: 「2の つぎは？」→「にの つぎは？」、「9の つぎは？」→「きゅうの つぎは？」、「10の まえは？」→「じゅうの まえは？」、「2と 4の あいだは？」→「にと よんの あいだは？」

## 検収基準（ユーザーが試聴で確認するポイント）

1. **読み崩れがないか**
   - 「ふたつ」「みっつ」「よっつ」「いつつ」「じゅっこ」等の促音が正しく発音されているか
   - 「ぞうきの はい」「いぶくろの い」が自然に区別できるか
2. **抑揚・速度が子供向けか**
   - 早すぎず、聞き取りやすい話速
   - 機械的すぎない自然な抑揚
3. **音量レベルが揃っているか**
   - 27ファイル間で音量差が大きくないか（後で正規化処理が必要なら指摘）
4. **ノイズ・無音時間**
   - 冒頭・末尾の無音が長すぎないか（理想は前後 100〜200ms 程度）
5. **カギ括弧除去 (Q95)**
   - 「カギカッコ おおきい カッコトジ」のような余計な読み上げが入っていないか
6. **同音異義の比較 (Q160)**
   - 通常版だけで「はい / い」が区別できるか、補足版の方が分かりやすいか
7. **数字の読み分け (Q24 vs Q170)**
   - Q24「2つ」→「ふたつ」、Q170「2」→「に」と文脈に応じた読み分けができているか
