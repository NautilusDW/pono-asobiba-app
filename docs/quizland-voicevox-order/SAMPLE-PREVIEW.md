# Quizland VOICEVOX 音声生成 発注書（サンプル版）

## 概要
- 発注先: Claude Cowork
- 生成エンジン: VOICEVOX
- 対象: フクロウ博士のクイズ（quizland）の問題文 + 4択選択肢を音声化
- 今回はフル版（全163問 × 5音声 = 815ファイル）の前のサンプル（21問 × 5 = 105ファイル）
- フル版発注前のフォーマット確認用
- 対象年齢: 5-6歳児（カタカナ読める前提）

## 出力仕様
- 形式: wav（48kHz / 16bit / モノラル）
- 命名規則: `q{問題番号2桁}_{パート}.wav`
  - パート: `q` (問題文) / `a` / `b` / `c` / `d` (選択肢)
  - 例: q01 の問題文 → `q01_q.wav`、q01 の選択肢A → `q01_a.wav`
- 出力ディレクトリ: （納品時に別途指示。ひとまず `audio/quizland/` を想定）
- 話者: 未定（VOICEVOX の話者選定はサンプル聞いてから決定）

## 読み方ルール

### 1. 色名の英語キー → 日本語
order_color カテゴリの選択肢は内部データでは英語キーで持っているが、画面表示・音声ともに日本語表記:

| 英語キー | 日本語（speech） |
|---|---|
| red | あか |
| blue | あお |
| yellow | きいろ |
| green | みどり |
| pink | ピンク |
| orange | オレンジ |
| purple | むらさき |
| cyan | みずいろ |

### 2. 数字 → 助数詞付き読み（count_total ほか）
裸の数字 (1〜10) は問題文の助数詞に合わせて読む:

| 数 | いくつ？ | なんこ？ | なんぼん？ | なんしょく？ |
|---|---|---|---|---|
| 1 | ひとつ | いっこ | いっぽん | いっしょく |
| 2 | ふたつ | にこ | にほん | にしょく |
| 3 | みっつ | さんこ | さんぼん | さんしょく |
| 4 | よっつ | よんこ | よんほん | よんしょく |
| 5 | いつつ | ごこ | ごほん | ごしょく |
| 6 | むっつ | ろっこ | ろっぽん | ろくしょく |
| 7 | ななつ | ななこ | ななほん | ななしょく |
| 8 | やっつ | はっこ | はっぽん | はっしょく |
| 9 | ここのつ | きゅうこ | きゅうほん | きゅうしょく |
| 10 | とお | じゅっこ | じゅっぽん | じゅっしょく |

### 3. 「0こ（ない）」のような括弧書き
質問が「なんこ？」「いくつ？」なら **「ぜろこ」** と読む（括弧の「ない」は無視）。

### 4. 既に助数詞付きの数字
データ側に「3こ」「5ほん」のように助数詞が付いているものは、画面表示はそのままだが、speech は読み崩れ防止のためカナで明示する:

- 「3こ」→「さんこ」
- 「5ほん」→「ごほん」
- 「6ぽん」→「ろっぽん」
- 「7しょく」→「ななしょく」
- 「8ぽん」→「はっぽん」
- 「7こ」→「ななこ」

### 5. 問題文中の数字も同様
問題文中の数字や読み崩れリスクのある語も、必要に応じて speech 側で読みやすく書き直す。

### 6. 句読点・記号
- 文末の「？」は残してOK（VOICEVOX が自然に処理する）
- 「、」「・」もそのまま
- 半角スペースは VOICEVOX の入力では自然なポーズになるので残す

## 確認事項（フル版発注前にユーザーに確認したい）

1. **話者**: VOICEVOX の話者は子供向けなので明るく親しみやすい声を想定。サンプルでは話者を仮選定してもらい、ユーザーレビュー後にフル版で確定。
2. **括弧（「」）の扱い**: opposite カテゴリでは speech から「」を除去した。フル版でも同方針でよいか。
3. **裸数字 vs 助数詞**: count_total の選択肢は画面表示が裸数字（2,3,4,5）だが speech は「ふたつ・みっつ・よっつ・いつつ」と読む。問題文が「なんこ？」「なんぼん？」「なんしょく？」の場合の読み分けはルール表に従う。
4. **単音節選択肢のアクセント**: Q160 のように「はい」「い」「め」「みみ」など短い語が並ぶ場合、VOICEVOX の話者によっては聞き分けにくい可能性あり。
5. **promptデータ側の表記揺れ**: 「3ぼん」「6ぽん」のような既存データの揺れは speech 側で正規化（「さんぼん」「ろっぽん」）した。
