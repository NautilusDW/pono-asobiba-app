# LP ゲーム紹介コピー改稿記録（2026-07-13）

## 1. 目的と現在地

`index.html` のゲーム紹介カードと詳細モーダルについて、次の問題を解消するためにコピーを見直した。

- 日本語として不自然な言い回しや、仕様書のような説明が混ざっていた
- `free` / `book` / `app` で利用範囲が異なる機能を、全員が使えるように読める箇所があった
- 保護者向け文章と子ども向け文章の文体が混ざっていた
- すべての遊びに同じ教育効果や「しっぱいしても だいじょうぶ」を当てはめ、内容が単調になっていた
- 実装済みのシールちょう画像・キャプションと記録内容が食い違っていた

この文書は、2026-07-13 の再レビュー後に採用したコピーと、2026-07-14 の実画面確認で追加調整した学びタグ・詳細導線の記録である。実装の正本は `index.html` とする。

## 2. コピーの方針

### 保護者向け

- `pc-play` は「何をする遊びか」を短く伝える
- `pc-growth` は遊び方を繰り返さず、その体験から得られる気づきや力を伝える
- `pc-chip` はモード名や難易度ではなく、その遊びで経験できる具体的な力を伝える
- 同じ `index.html` が複数の tier で表示されるため、特定 tier だけの機能を共通機能のように断定しない
- 機能数や判定段階など、理解に不要で変更されやすい数字は並べない
- 教育効果を大きく言い切らず、実際の操作や選択とつながる表現にする

### 子ども向け

- 詳細モーダルのキャプションは、ひらがな・カタカナを基本にする
- 画像で見えていること、実際にできることを具体的に説明する
- 自由な演奏や配置を「しっぱい」と評価しない
- 安心感は「もどれる」「えらびなおせる」「たしかめられる」のように、実際の仕組みで伝える
- 未公開機能や将来の追加を期待させる表現は入れない

## 3. 採用コピー

### いろどりべんとう

**カード：遊びの説明**

好きなおかずを選び、タップでおべんとうを作る遊び。のりの顔パーツも動かして、自分らしく盛りつけられます。

**カード／モーダル：ココが育つ**

おかずの色や形を見比べ、のりの顔パーツを動かしながら仕上げます。**組み合わせを考える力と、食への興味**が育ちます。

**学びタグ**

見比べる力／組み合わせる力／食への興味

**場面キャプション**

1. ごはんを つめて、 すきな おかずを いろどりよく ならべよう。
2. えほんの あいことばや アプリで あそべる「おねがいモード」では、 どうぶつの おきゃくさんの おねがいに あわせて、 おべんとうを つくるよ。
3. すきな おかずや のりを ならべて、 じぶんだけの おべんとうを つくれるよ。

### ランタンの めいろ

**カード：遊びの説明**

夜の森で迷子になった動物を助けに行く迷路あそび。ランタンを頼りに道を選び、途中のしかけや謎を解きながら進みます。

**カード／モーダル：ココが育つ**

分かれ道で進む道を選び、途中のしかけやミニゲームに挑戦。別の道を試す体験を通じて、**考え直す力と最後まで取り組む力**が育ちます。

**学びタグ**

道を選ぶ力／考え直す力／やりぬく力

**場面キャプション**

1. ランタンの あかりを たよりに、 よるの もりへ。 ポノと いっしょに、 まいごの どうぶつを さがそう。
2. みちを ふさぐ しかけに であったら、 じゃんけんや クイズなどの ミニゲームに ちょうせんしよう。
3. いきどまりでも、 ひとつ まえの わかれみちに もどれるよ。 べつの みちを ためして、 どうぶつを たすけよう。

### ポノの おとタッチ

**カード：遊びの説明**

ボタンをタップして、ドレミや動物の声を鳴らす音あそび。好きな音を重ねたり、曲に合わせてタイミングよく鳴らしたりできます。

**カード／モーダル：ココが育つ**

好きな音を鳴らしたり、曲に合わせてタイミングよくタップしたりする中で、**音の違いに気づく力とリズム感**が育ちます。

**学びタグ**

音を聞き分ける力／音を組み合わせる力／リズム感

**場面キャプション**

1. ボタンを おすと、 ドレミや どうぶつの こえが きこえるよ。 すきな おとを ならしてみよう。
2. 「じゆうモード」では、 すきな おとを くりかえしたり、 まぜたりできるよ。 じぶんだけの えんそうを たのしもう。
3. 「リズムモード」では、 きょくに あわせて おとを ならそう。 タイミングよく おとを ならしつづけると、 あいての ひょうじょうが かわるよ。

### ポノの なかよしパズル

**カード：遊びの説明**

ピースを合わせて絵を完成させるパズル。遊びを進めると「なかま」が増え、選んだ「なかま」によって助け方や挑戦のルールが変わります。

**カード／モーダル：ココが育つ**

少ないピースから始め、慣れてきたら少しずつ難しいパズルに挑戦します。**試しながら進め方を工夫する力と、最後まで取り組む力**が育ちます。

**学びタグ**

形を見分ける力／工夫する力／やりぬく力

**場面キャプション**

1. まずは よっつの ピースから。 パズルを すすめると、 どうぶつの 「なかま」を えらべるように なるよ。
2. 「キツネ」は えらんだ ピースを おく ばしょを みせてくれるよ。 「こじか」と いっしょなら、 ピースが ただしい ばしょに ちかづくと ひかるよ。
3. どこに おくか まよったら、 「みる」ボタンで できあがりの えを たしかめられるよ。 じっくり ためしてみよう。

### フクロウはかせの なぞなぞ

**カード：遊びの説明**

フクロウはかせの問題に挑戦。手がかりを見たり聞いたりしながら、答えを考えるクイズあそびです。

**カード／モーダル：ココが育つ**

色や数、ことばなど、いろいろな問題に挑戦。迷ったときはヒントを見ながら、**手がかりをもとに考える力**が育ちます。

**学びタグ**

見る・聞く力／手がかりを使う力／答えを考える力

**場面キャプション**

1. もりの きょうしつで、 フクロウはかせの なぞなぞに ちょうせんしよう。 いろや かずなど、 いろんな もんだいが でてくるよ。
2. こたえに まよったら、 はかせの ふきだしを タップして ヒントを みよう。 まちがえても、 すぐに えらびなおせるよ。
3. フクロウはかせの おうちは、 もりの おくに あるよ。 なかへ はいって、 なぞなぞに ちょうせんしよう。

### ガチャガチャとお店

**詳細導線**

シールの集め方・選び方を詳しく見る

**カード／モーダル：ココが育つ**

ガチャで偶然出会う楽しさと、お店で選ぶ楽しさがあります。まだ持っていないシールがあるときは、その中の一枚をどんぐりと交換するか「とりおき」するか考え、**見通しをもって選ぶ力**が育ちます。

**場面キャプション**

1. カプセルを あけると、 まだ もっていなかった シールか、 もう もっている シールかが わかるよ。
2. シールの おみせ「こもれびや」では、 どんぐりで すきな シールを えらべるよ。 まだ もっていない シールが あるときは、 おみせが そのなかから いちまい えらんで ならべてくれるよ。 その シールは 「とりおき」も できるよ。

### シールちょうとミュージアム

**詳細導線**

シールの貼り方・飾り方を詳しく見る

**カード／モーダル：ココが育つ**

絵本のようにページをめくり、シールを好きな場所に貼ったり、絵を描き足したりできます。**配置を工夫する力と、表現する力**が育ちます。

**場面キャプション**

1. 「シールちょう」は えほんの ように ページを めくれるよ。 シールは すきな ばしょへ なんどでも うごかせるよ。
2. あつめた シールを きんいろの がくぶちに かざって、 ひとつずつ じっくり ながめられるよ。 まだ あつめていない シールは、 かげに なって みえるよ。
3. 「シールミュージアム」の テーマべつの へやを まわって、 あつめた シールを さがしてみよう。

## 4. 実装確認で反映した事実

- 弁当の実際のモード名は「おねがいモード」「じぶんでつくる」。ただし注文遊びや一部ののり素材は tier によって利用範囲が異なるため、共通カードではモードを断定していない
- 迷路の障害には虫以外も含まれ、遊べるステージ数も tier で異なるため、「9種のおじゃまムシ」は使用していない
- おとタッチの太鼓は全 tier 共通ではないため、共通説明はドレミと動物の声を中心にした
- パズルの「なかま」の種類は tier で異なり、お絵かきパズルには利用条件があるため、種類数とお絵かき機能を共通訴求から外した
- なぞなぞの難しさと問題範囲は tier で異なるため、「3段階」「8テーマ」といった数を共通訴求から外した。日ごとの上限がある報酬も共通キャプションでは断定していない
- 「こもれびや」の未所持シール保証には、未所持シールが残っていることという条件があるため、保証を断定する説明は使用していない
- シールちょうは Three.js による3Dページめくりであり、物理エンジンによる「本物の物理演算」とは表現していない

## 5. 変更と確認の扱い

- コピー変更は `index.html` のカード本文と `GAME_MODAL_DATA` の両方へ同じ内容を反映する
- HTML変更のため、`sw.js` の `CACHE_VERSION` を更新する
- モバイルとデスクトップでカードの高さ、改行、モーダルのスクロールを確認する
- シールちょうの3枚は現行画像と復元済みキャプションを維持する
- 画像そのものの追加制作は今回のコピー改稿には含めない
