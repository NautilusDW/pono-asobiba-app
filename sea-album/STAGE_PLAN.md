# ポノの海底アルバム ステージ計画

## 基本ループ

1. ステージを選ぶ。
2. ショップで買った永続装備は、ニューゲームや再挑戦でも最初から付いた状態で始まる。
3. えさ魚雷と泡つきえさボムで生き物と仲良くなり、ステージごとの仲良し度を上げる。
4. 仲良し度のヒントを見ながら、そのステージのキーアイテムを探す。
5. 仲良し度が100になるとボスが出る。
6. キーアイテムを使ってボスを落ち着かせ、最後にえさをあげてアルバムに登録する。
7. コインでショップ装備を増やし、未発見カードとレアカードを集めるために再挑戦する。

## パワーアップとショップ

基本方針は、「ステージ内で拾うものはそのプレイ中だけ強くなる」「ショップで買うものは次のプレイ以降も残る」に分ける。

- 通常ステージのパワーアップ: そのプレイ中だけ有効。敵と仲良くなる、宝箱を開ける、隠し部屋を見つけるなどで出る。
- ショップのアップグレード: コインで買う永続装備。一度買うと、ニューゲームや再挑戦でも最初から付いた状態で始まる。
- 重要: ショップで「強さそのもの」を全部売らない。ショップは上限や便利さを広げ、ステージ内の拾い物でその場の成長を感じる。
- ボス用のキーアイテムはショップで売らない。毎回そのステージ内で見つける。
- 子ども向けの本編では永続装備を常時オンにする。大人向けの手ごたえが必要になったら、将来「そうびなしチャレンジ」を別モードで足す。

### 通常ステージで上がるパワー

- パワー1: えさ魚雷1本。画面内に1本だけ出せる。
- パワー2: えさ魚雷1本のまま、ホーミングが少し強くなる。
- パワー3: えさ魚雷の発射管が2本になる。連射ではなく、1本ずつ大事に撃つ魚雷感を残す。
- パワー4: えさ魚雷の発射管が3本になる。泡つきえさボムの範囲も少し広がる。

パワーアップの出し方:

1. 初めて仲良くなった生き物は、小さいパワー星かコインを出す。
2. 仲良し度25、55、85の節目でパワー星を出す。
3. 宝箱や寄り道では、少し大きいパワー星を出してもよい。
4. ボス前にパワーが足りない場合だけ、救済用に1つ出す。

### ショップで買う永続装備

ショップはステージ間の「じゅんび画面」に置く。ステージ中に買わせると流れが止まるので、ステージ中は拾うだけにする。

- 発射管キット: えさ魚雷の最大本数上限を増やす。最初は最大2本まで、後半で最大3本まで。
- 磁石の貝: 近くのコインを吸いやすくする。
- 小さなライト: 暗い横穴や深い場所で、隠れた生き物や宝箱を見つけられる。
- 地図ヒント: まだ開けていない宝箱や戻りポイントを、画面端ガイドで少し出しやすくする。
- ボムポーチ: 泡つきえさボムの同時数や範囲の上限を増やす。
- JETクーラー: JETの再使用時間を少し短くする。
- スタート強化: ステージ開始時にパワー2から始められる。ただし最初から最大火力にはしない。
- なかよしノート: 一度見たヒントをステージ開始時から読み返せる。次のヒントも少し早く出る。

### ショップ専用にするもの

ステージ内で拾えない、ショップでしか手に入らないものは「探索の質が変わるもの」にする。

- 小さなライト。
- 地図ヒント。
- 磁石の貝。
- 発射管キットの最大上限解放。
- JETクーラー。
- なかよしノート。

ステージ内で拾えるものは「その場の強さが変わるもの」にする。

- そのプレイ中のパワー1〜4。
- 一時シールド。
- コイン。
- その場だけ見えるヒント泡。
- ボス用キーアイテム。

### 仲良し度とボス解放

各ステージには仲良し度を0〜100で持たせる。敵を倒すのではなく、生き物にえさをあげて「観察できた」「喜んで帰った」時に上がる。

- 初めて仲良くなった種類: +18。
- 同じ種類ともう一度仲良くなった時: +4。
- 宝箱、救出、スイッチなどの小ギミック達成: +10〜15。
- ボス用キーアイテム入手: +12。

仲良し度100になるとボス警告が出る。ただし、ボス用キーアイテムがない場合は、すぐボス戦に入らず「このままだと危ない。〇〇を探そう」と中央通知と画面端ガイドで戻す。キーアイテムがある場合だけ、ボス戦へ進む。

ヒントは仲良し度の節目で少しずつ具体的にする。画面端で一瞬言わせず、中央カードと上部目的表示で必ず見えるようにする。

- 20: そのステージのボスがどんな動きをするかを短く見せる。
- 40: キーアイテムが入っている場所の種類を教える。「大きな貝があやしい」など。
- 60: その場所を開けるために必要な相手や道具を教える。「カギはヤドカリが知ってる」など。
- 80: ボス対策を教える。「すなだんごがあれば、突進を止められる」など。
- 100: ボス警告。キーアイテムがあればボスへ、なければ取りに戻す。

この仕組みにすると、プレイヤーは仲良し度を上げるために動き回りながら、飽きる前に次の目的を受け取れる。ショップ装備はその移動や発見を助けるが、ボス突破そのものはステージ内のキーアイテムで行う。

## ステージ1 探索ギミック

ステージ1では、メニューの多い探索ゲームにせず、「先へ進む → 何かを見つける → 左へ戻る → 宝箱を開ける」だけを自然に覚えられるようにする。上下スクロールは将来ステージ用に残し、1面は左右スクロールだけで成立させる。

- ワールド幅: 1面は広げすぎず、スタート近くの貝宝箱と少し進んだ先のヤドカリの間にだけ探索の余白を作る。
- ボス条件: 時間経過では出さず、仲良し度100 + 貝宝箱から「カブトガニのすなだんご」を取った状態で右奥へ進むと、カブトガニ警告へ進む。
- 足りない条件がある場合: 右奥でボス戦へ入れず、中央カードと上部目的表示で「カギ」「貝宝箱」「仲良し度」「すなだんご」のどれが足りないかを順番に見せる。
- 見せ方: 重要な取得物は魚の吹き出しだけに頼らず、中央の獲得カード、上部の目的表示、画面端の矢印ガイドで必ず見えるようにする。
- レイヤー: アルバム登録、コイン獲得、吹き出しなどの文字は潜水艦より前面に出し、プレイヤーに隠れないようにする。
- 間の作り方: 開始直後の目の前の敵に鍵を持たせない。まず普通に浅瀬を進ませ、少し奥でヤドカリに会ってから鍵をもらう。
- ガイド整理: 単なる移動方向には丸マーカーを出さない。「右へ進む」は上部目的表示だけにし、画面内の丸はヤドカリや貝宝箱など実体のある重要対象に限る。
- セリフ整理: 吹き出しはヤドカリとボスなど重要キャラだけ。通常の生き物は吹き出し箱を使わず、濃い文字に白い縁取りを付けて短く出す。
- 表情: 通常の生き物は最初は怒り顔。1発目はまだ怒り、2発目で「あれ？」、最後に喜び顔とハートで帰る。

1. スタート近くの貝宝箱
   - x=520 あたりに、最初から閉じた貝を見せておく。
   - 近づくと「まだ開かない。右でカギを探そう」とだけ伝える。

2. 少し奥のヤドカリ
   - 開始直後ではなく、少し右へ進んだ先で出会う。
   - えさをあげて仲良くなると、「貝のカギ」をくれる。
   - ここで中央カードと矢印ガイドを出して、左へ戻る意味を見せる。

3. 貝宝箱を開ける
   - 左へ戻って閉じた貝に触れると開く。
   - コインと「カブトガニのすなだんご」が出る。
   - コインがゆっくりばらまかれ、キーアイテムを取った達成感を中央カードで見せる。
   - その後は右奥へ進む導線だけ出し、カブトガニの警告演出へ進む。

4. カブトガニ戦
   - すなだんごを持っていると、カブトガニの突進後に少し止まる。
   - 止まったところへえさ魚雷を当てると、怒りから「あれ？」へ変化する。
   - 最後は喜び吹き出しとハートで帰り、アルバムに登録される。
   - すなだんごを持っていない場合は本戦へ入れず、「貝宝箱に戻ろう」と中央通知で戻す。

5. 仲良し度ヒント
   - 20: 「カブトガニは まっすぐ つっこんでくる」
   - 40: 「左の 大きな貝に いいものが あるかも」
   - 60: 「貝のカギは ヤドカリが しってる」
   - 80: 「すなだんごが あれば つっこみを とめられる」
   - 100: 「カブトガニが くるよ。すなだんごを もった？」
   - POWER星は仲良し度25/55/85で出す。POWER3で画面内のえさ魚雷が2本、POWER4で3本まで増える。

6. コインの使い道
   - 最初は攻撃力ではなく、探索が楽しくなる便利アイテムに使う。
   - 候補は「小さなライト」「磁石の貝」「地図ヒント」。
   - えさを読む時間を保ちつつ、繰り返し探索する意味を作れる。

## 見た目メモ

- 生き物が最後に喜ぶときは、吹き出しをピンクにしてハートを出す。
- 喜んだ生き物とハートは同じタイミングで薄くなって消える。
- コインは平面の丸ではなく、3D風にくるくる回りながら、ゆっくり海底へ落ちる。

## エフェクト素材方針

- 泡、小さいきらめき、ヒットリング、水のゆらぎなどの軽いエフェクトは、画像ではなくゲーム画面上で直接描く。
- 貝宝箱の開封だけは、別PNGの星エフェクトを散らし、その上に必要なら Canvas 側で軽いグロウを足す。
- キャラクターのような特別なデザインが必要な場合を除き、汎用の泡エフェクト画像は生成しない。
