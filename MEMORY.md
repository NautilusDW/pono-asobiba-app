## Key Learnings

### 単一HTMLファイルWebアプリ設計パターン
- Web Audio APIを使えば外部音声ファイルなしで効果音を生成可能
- `window.KidsTraceApp` のようなパブリックAPIを用意しておくと将来のAIキャラクター統合が容易
- Google Fonts `Zen Maru Gothic` は子ども向けの丸みフォントとして適切

### 文字なぞり・書き順ガイド
- **書き順データの自作は非推奨** → Hanzi Writer + 既存データソースを使う方が圧倒的に精度が高い
- ひらがな/カタカナ: `ailectra/kana-json` (animCJK由来、HanziWriter互換JSON)
  - CDN: `https://cdn.jsdelivr.net/gh/ailectra/kana-json@v0.0.1/data/{char}.json`
- 漢字: `hanzi-writer-data-jp` (KanjiVG由来)
  - CDN: `https://cdn.jsdelivr.net/npm/hanzi-writer-data-jp@0/{char}.json`
- Hanzi Writer v3.5: `charDataLoader`オプションでカスタムデータ読み込み可能
- `writer.quiz()` でなぞり判定、`animateCharacter()` でお手本アニメーション
- アルファベット・数字はHanzi Writer非対応 → Canvas fallbackが必要
