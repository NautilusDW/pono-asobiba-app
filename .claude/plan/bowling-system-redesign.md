# ゲームシステム根本リデザイン — リサーチ統合プラン

## リサーチから見えた「中毒性のあるゲームシステム」の共通構造

```
中毒ループ = シンプルな入力 → 物理による予測不能な展開 → 連鎖的な報酬 → もう1回
```

最も重要な発見: **スイカゲーム（Suika Game）** が今回の条件に完璧にフィットする。
- 円の物理衝突のみ（既存エンジンがそのまま使える）
- 3歳でも遊べる（タップで落とすだけ）
- 大人もハマる（戦略的配置＋連鎖の快感）
- 2023年に8000万人がプレイした実績

## 提案A: スイカゲーム型マージパズル（★推奨）

### コンセプト
「びっくりボウリング」のピン画像をそのまま使い、**ピンを落として同じピン同士がくっついたら大きくなる**マージゲーム。

### コアループ
1. 画面上部からピンを1個ずつ落とす（横位置だけ指定）
2. 同じ種類のピンが触れ合うと → 合体して次のサイズのピンに進化
3. 容器がいっぱいになったら → 花火が上がってリセット（ゲームオーバーなし）
4. 連鎖で大量合体 → 画面が揺れる＋音が上がる＋パーティクル爆発

### 3歳向けアレンジ
| 本家スイカゲーム | うちのアレンジ |
|---|---|
| 11段階のフルーツ | 5段階のピン（既存の13種から5つ選抜） |
| 小さな容器 | 画面幅いっぱいの大きな容器 |
| ゲームオーバー→やり直し | 満杯→花火→自動リセット（罰なし） |
| ランダムな次のピース | 2-3種のみ（選びやすい） |
| 静かな合体 | 派手な合体エフェクト＋音＋揺れ |
| スコア重視 | 合体のたびにお祝い演出 |
| 物理が荒ぶる | 重力弱め＋摩擦強め（安定しやすい） |

### 大人が楽しめるポイント
- 5段階でも配置戦略は深い（同じ種類を近くに集める）
- 物理の予測不能さが連鎖を生む（スイカゲーム同様の快感）
- 最大サイズのピンを2つ合体→消滅＝スペース確保の達成感
- スコアは表示（ハイスコア保存）、でも押し付けない

### 技術的フィット
- **既存の物理エンジンがそのまま使える**（circle-circle collision）
- ピン画像13種のうち5種を段階的サイズで使用
- 容器 = 壁3枚（左、右、底）の静的境界
- 合体判定 = collision時にtypeが同じかチェック→同じなら2体を削除＋1体を新規生成

### 実装ステップ
1. 容器（箱）の描画＋壁判定
2. タップで横位置を指定→ピンを落下
3. 同種ピン接触→合体（次のサイズに進化）
4. 5段階のピンサイズ定義（半径 + 画像マッピング）
5. 合体時エフェクト（パーティクル＋音のピッチ上昇＋画面揺れ）
6. 満杯検知→花火→リセット
7. スコア＋ハイスコア（localStorage）
8. 「次のピン」プレビュー表示

## 提案B: ピンコ（Plinko）型 — 落として見守る

### コンセプト
画面にペグ（杭）がびっしり配置されていて、上からボールを落とすとカチカチ跳ねながら下に落ちる。下にはスコアゾーンがある。

### コアループ
1. タップした横位置からボールを落とす
2. ボールがペグに跳ねながら落下（物理演算）
3. 画面下のスコアゾーン（100, 500, 1000, 500, 100）に到達→得点
4. 連続で同じゾーンに入ると倍率UP
5. ボールは何個でも同時に落とせる（タップ連射OK）

### 3歳向け
- タップするだけでボールが出る
- カチカチ跳ねる音が楽しい
- カラフルなボールがいっぱい画面にある
- ゲームオーバーなし

### 大人向け
- 落とす位置の微調整でスコアが変わる
- 確率と物理の読みがある
- パチンコ的な快感

### 弱点
- 見守る時間が長く、能動的操作が少ない
- 3歳には「見てるだけ」になりがち

## 提案C: チェインリアクション（Boomshine型）

### コンセプト
画面中にカラフルなボールが飛び交っている。タップすると爆発が起きて、近くのボールも巻き込んで連鎖爆発。

### コアループ
1. 画面にボールがたくさん浮遊
2. タップ→その場所で爆発（広がる円）
3. 爆発に触れたボールも爆発（連鎖）
4. 連鎖数 = スコア
5. 連鎖が途切れたら新しいボールが湧いてくる

### 3歳向け
- タップだけ。どこを押しても何か起きる
- 連鎖の「ポポポポ」という音が楽しい
- 常にボールが動いてて見てるだけでも楽しい

### 大人向け
- タイミングと位置の最適化（ボールの密集地点を読む）
- 目標: 全ボール一撃（理論上可能だが超難しい）

### 弱点
- 操作が「タップ1回」だけなので飽きやすい可能性
- 3歳にはちょっと抽象的

## 提案D: ハイブリッド（現行おもちゃ + マージ）

### コンセプト
今のタップでボール発射 + ピンが降ってくるシステムを残しつつ、**ピン同士がぶつかったら合体する**マージ要素を追加。

### コアループ
1. ピンが上から降ってくる（現行）
2. タップでボール発射（現行）
3. ボールがピンに当たる→ピンが弾き飛ぶ（現行）
4. **NEW: 同じ種類のピンがぶつかると合体して大きくなる**
5. 大きくなったピンをさらにぶつけて連鎖合体
6. 最大サイズに達したピンは爆発して消える＋大量パーティクル

### 3歳向け
- タップするだけでボールが出る（現行通り）
- ぶつけて合体する楽しさが追加
- 爆発演出が派手

### 大人向け
- 同じピンを狙って当てる戦略性
- 連鎖合体を狙うスキル要素
- 現行の「触るだけで楽しい」＋マージの深み

### メリット
- 現行コードからの変更が最小
- ボウリングの延長として自然

## 推奨: 提案A（スイカゲーム型）

### 理由
1. **実績**: スイカゲームは8000万人が遊んだ。このシステムは「楽しい」ことが証明済み
2. **技術的フィット**: 既存の円物理エンジンがそのまま使える
3. **全年齢**: 3歳は「落として合体を見る」、大人は「配置を考えて連鎖を狙う」
4. **既存資産**: ピン画像13種がそのまま使える
5. **爽快感**: 連鎖合体の瞬間が最も爽快（リサーチで最高評価のゲームシステム）
6. **リプレイ性**: 物理の不確定性で毎回違う展開

### ピン進化チェーン（5段階案）
```
pin_0 (極小 r=20) → pin_3 (小 r=30) → pin_6 (中 r=42) → pin_9 (大 r=56) → pin_12 (特大 r=72)
```
特大同士が合体 → 消滅 + 大爆発エフェクト + ボーナススコア

### Risks and Mitigation
| Risk | Mitigation |
|------|------------|
| スイカゲームの「コピー」に見える | ピンのキャラクター性＋子供向け演出で差別化 |
| 5段階では浅すぎる | まず5段階で出して反応を見る、後で増やせる |
| 物理の安定性（積み重なり） | 摩擦強め＋重力弱めで安定させる |
| タッチ精度（3歳の太い指） | 落下位置をグリッド的にスナップ or ゾーン分け |
