# Hide Seek Creatures asset sources

All runtime assets in this folder are local to the native Unity game. The game
does not fetch visual or audio files from the network.

## Stage `km01`

- `forest_background.png`
  - New GPT Image 2 artwork generated for batch
    `1174-hide-seek-creatures` on 2026-07-10.
  - Raw delivery:
    `tmp/alpha_pending/1174-hide-seek-creatures/hide_seek_forest_bg_day_raw.png`
  - Prompt intent: a bright 16:9 watercolor picture-book forest clearing with
    readable left, center, and right hiding places; no creatures, fog, UI, or
    text.
- `rabbit.png`
  - Reused from `assets/zukan/animals/flower_path/rabbit.png`.
- `fawn.png`
  - Reused from `assets/zukan/animals/sunlit_forest/fawn.png`.
- `hedgehog.png`
  - Reused from `assets/zukan/animals/mushroom_forest/hedgehog.png`.
