# Review D1 (independent, difficulty-focused)

## Overall verdict
- **部分採用** — Lv1 新作 8問は概ね優秀で採用可。ただし Lv3 新規追加 2問に懸念あり（特に「カバの汗」は detail のニュアンスが幼児向けに不適切）、また「タコのしんぞう」の Lv2 降格と総枠調整で実装者自身が結論を保留している部分は明確化が必要。

## NEW Lv1 question-by-question

- **Lv1 #1 「バナナは じゅくすと なにいろに なる？」**: **妥当**。3yo でも家庭で経験する事実、4択もすべて色名のみで認知負荷低い。詰問なし。
- **Lv1 #2 「ねこは なんて なく？」**: **妥当（やや過易）**。3yo にとって完全に既知レベル。Lv1 として OK だが、ほぼ全員正解する可能性。distractor「コケコッコー」は明確に違うので問題なし。
- **Lv1 #3 「ぞうの からだで いちばん ながいのは どこ？」**: **妥当**。絵本・図鑑頻出。「いちばん ながい」という比較概念は 3yo にはやや抽象的だが、「ぞうの はな」の知識は普及しているので OK。
- **Lv1 #4 「うしの なきごえは？」**: **妥当（やや過易・#2と同型）**。鳴き声問題が連続している点は theme variety で要注意（#2と#4が両方鳴き声）。Lv1 で 2問続けるのは許容範囲だが、片方を別系統に差し替える余地あり。
- **Lv1 #5 「たまごから うまれる いきものは どれ？」**: **妥当**。にわとり=卵 は普及知識。ただし厳密には魚も虫も卵から生まれるが、選択肢が「いぬ・うし・ぶた」と全て哺乳類なので 3yo の実感には合致。OK。
- **Lv1 #6 「りんごは なにいろ？」**: **妥当（やや過易）**。「みどりの りんごも あるんだ」と detail でフォローしてある点は good。
- **Lv1 #7 「おひさまが でてくるのは どっち？」**: **妥当**。生活実感に直結。distractor「まよなか・ゆうがた」は紛らわしさがあるが、「あさ」が明確正解。
- **Lv1 #8 「うさぎの みみは どんな かたち？」**: **やや過難ぎみ**。文章での形状記述（「まるくて みじかい」「ぺたんと たれている」）が 3yo にはやや読解負荷。ただし絵本接触で「うさぎ=長い耳」は既知。Lv1 の中では最も難しい。許容。

→ 8問とも概ね Lv1 として妥当。**懸念は鳴き声問題が2問（#2 #4）連続しテーマ重複している点のみ**。

## Implementer deletion decisions

- **ハヤブサDNA**: **同意**。「DNA」という単語が 3-6yo に通じない。事実は正しいが obscure すぎる。削除妥当。
- **ハチ8の字ダンス**: **同意（ただし惜しい）**。「8の字」「ダンス」は幼児にも届く可能性あり、figure ある絵本もある。しかし4択 distractor「なく/においを つける/とんで みせる」がどれも「ありそう」で 6yo には推測困難。削除に賛成だが、将来「ダンスする」の選択肢を残しつつ別形式（detail で von Frisch を語らない）で再登場させる余地あり。
- **タコイカの青い血**: **同意**。「血」というワードは 3-6yo の一部にショック・嫌悪を与える可能性。さらに detail の「てつで あかい / どうで あおい」は化学概念で 6yo に届かない。削除妥当。

## Implementer addition decisions

- **フラミンゴ ピンクの理由**: **妥当（Lv3 として適切）**。「たべものの 色が からだに うつる」は驚き fact として優れる。hint も適切。distractor「そらの いろを すいこむ／ペンキを ぬる」は 6yo に魅力的な誤答候補で良問。**採用推奨**。
- **カバの汗 赤ピンク**: **過難＋やや怖い**。問題自体は driver として面白いが、以下が懸念:
  1. detail の **「ひやけどめに なるよ」** は OK だが、「カバの あせは あかピンク」を一文目で言うと、絵本的経験のない子は「血が出てる？」と誤解する可能性あり。読み聞かせる親が「血じゃないよ」と説明する必要が生じる。
  2. 「あせ」と言いつつ実際は粘液で、これを「あせ」と呼ぶこと自体は許容だが、Lv3 にしては「びっくり度」が高すぎる/恐怖系に倒れやすい。
  3. 削除した「タコイカの血」と同種の「色つき体液」trivia なので、削除根拠との整合性に弱さあり。
  → **修正案**: detail の冒頭を「カバの からだから でる ピンクの えきは…」に変えて「あせ」感を弱め、または「ひやけどめ」をもっと前面に。または別の Lv3 候補（例: ペンギンの卵をオスが温める／カメレオンが色を変える理由）に差し替え検討。

## Theme variety in Lv1
- **どうぶつ**: 5問（ねこ・ぞう・うし・にわとり・うさぎ）
- **たべもの**: 2問（バナナ・りんご）
- **しぜん**: 1問（おひさま）
- **その他**: 0問

→ **動物に偏重（5/8）**。特に **#2 ねこ鳴き声 + #4 うし鳴き声 が連続的に「鳴き声」テーマで重複**。CLAUDE.md レビュー観点の「6 animal questions in a row 禁止」は守れているが、5問は多め。

**修正案**: うさぎ #8 または うし #4 を「のりもの」「いろ（虹）」「天気」系の問題に差し替えて 動物4 / たべもの2 / しぜん1 / その他1 のバランスへ。例:
- 「あめの ひに そらに でる いろの はしは なに？」→ にじ（OK 3yo にも届く）
- 「ふゆに そらから ふってくる しろい ものは？」→ ゆき

## Recommended deltas

1. **Lv1 #2 と #4 の鳴き声重複を解消**: #4「うしの なきごえ」を「あめの ひに でる いろの はしは？（にじ）」または「ふゆに そらから ふる しろいものは？（ゆき）」に差し替え。これで自然テーマ確保。
2. **Lv3「カバの汗」の detail 緩和**: 「カバの あせは あかピンク」を「カバの からだから でる ピンクの えきは ひやけどめに なるよ。ちが でてるんじゃ ないよ！」など、「血じゃない」を明示するフォローを追加。または別問題に差し替え。
3. **Lv3「タコのしんぞう」は Lv3 のまま維持**: 降格すると枠調整が複雑化し、実装者自身も決め切れていない。Lv3=8 を維持し、Lv2=8 のまま動かさないのが運用上シンプル（フラミンゴ・カバ追加を入れて Lv3=8 を達成）。
4. **Lv3「キリン 1日 2じかん」の正解値再検討**: 「3-5じかんくらい」に変更し detail も合わせる。または本問を保留してフラミンゴ・カバ・タコしんぞうを Lv3 に置けば穴は埋まる。
5. **Lv1 #8「うさぎの耳」の選択肢簡略化**: 「ながい」「みじかい」「たれている」「ちいさい」など 1語のみに短縮し読解負荷を下げる。
6. **総枠の最終確定**: Lv1=8 / Lv2=8 / Lv3=8 で Lv3 = (維持5) + (フラミンゴ + カバ) + (タコしんぞう or キリン差し替え1) = 8 とすれば、実装者の枠調整保留が解消する。
