## Issues found

### Issue 1 (Critical conceptual): Lv3 #5「ちょうほうけいと しかくの ちがい」
**問題**: タスク指示書でも reviewer focus に挙げられた論点。
- 「しかく」は元来「四角(quadrilateral)」または「四角形」として、長方形・正方形・台形・菱形すべてを含む広義の語として使われることが多い。
- しかし日本の幼児向け教材では「しかく」=「正方形」として扱われることがほとんど。
- 提案の答え「たてとよこのながさがちがう」は「しかく=正方形」前提でないと成立しない (正方形は縦=横、長方形は縦≠横)。
- 一方、Lv1 #2 で `square` を「しかく」と呼び、Lv2 #1 で `rectangle` を「ちょうほうけい」と呼ぶ用語体系を採用しているので、内部的には「しかく=正方形」で一貫している。
- **結論**: 内部一貫性は保たれており致命的ではないが、「しかく」の意味が広義 (四角形全般) の場合に答えが成立しないため、Lv3 #5 は曖昧さを残している。ただし他の選択肢 (「かどのかずがちがう」「まるい」「いろがちがう」) と比べれば、「たてとよこのながさがちがう」が最も妥当な答えなので、差し戻すほどではない。**△ 維持可**。

### Issue 2 (Minor): Lv3 #7 「ちょうほうけいの かどは 4こ」と Lv3 #4 「square の かど 4こ」の重複感
- Lv3 #4 (square 角=4) と Lv3 #7 (rectangle 角=4) は答えがどちらも 4こ で同じ。Lv3 #8 (diamond 角=4) も含めると Lv3 8 問中 3 問が「4こ」答え。
- 数値の偏りが学習効果を損なう可能性。1問は「いちばんかどがおおいかたちはどれ?」のような変則問題に置換するか、選択肢の並び順を変えて区別を促したほうがよい。
- **推奨**: 維持しても致命的ではないが、reviewer note 2 とも合致する。

### Issue 3 (Minor): Lv3 distractor「おこる」が「しかる」とほぼ同義
- ほめる⇄しかる の distractor「おこる」は「叱る」と意味的にかなり近い (むしろ叱る≒怒る)。子供がどちらでも丸と感じる可能性。
- **推奨**: distractor の「おこる」を別語 (例:「だきしめる」「あやまる」「うなずく」など) に差し替えるとより明瞭になる。**現状維持でも可**。

### Issue 4 (Minor): Lv1 `oval` のヒラガナ「たまごがた」と Lv2 `oval` の問題重複
- Lv1 #7 (oval, answer=たまごがた, 4択) と Lv2 #5 (oval, answer=たまごがた, 4択) は問題文も `q:'これは どんな かたち？'` で同一。選択肢の並び順だけが違う (Lv1: 0番目, Lv2: 3番目)。
- 厳密には「重複」(q+shape+answer文字列が同じ) だが、出題形式上はランダムスケジューラが回しているなら多様性として許容範囲。
- **推奨**: Lv2 oval の問題文を「これは なんて よぶ かたち？」など微変更すれば回避できる。**現状維持でも可**。

### Issue 5 (Minor): Lv1 `diamond` (ひしがた) の難易度
- reviewer note 1 でも触れられている通り、「ひしがた」という名称は 3 歳児にはまだ難しい。ただし他選択肢 (まる/ほし/ハート) と図形的に大きく異なるので消去法で回答可能。
