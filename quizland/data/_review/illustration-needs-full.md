## ① order_color (色問題) — 全24問 (questions.js L14-68)

### 共通方針
- 現行: CSS ベタ塗りチップ。色名と位置の理解は出来るが「絵」としての温かみがない。
- 推奨: 色付きの「もの」(風船、積み木、旗、おもちゃ、フルーツ等) を一列に並べたイラスト。
- 選択肢も同じ「もの」の単体絵 (色違い) で構成すると「色」と「もの」の同時記憶が効く。
- 統一テーマ案: **「ポノが N 個の色付き風船を持って一列に並べている」** → 風船の色がそのまま items の色配列になる。
