### 最大の懸念点 (TOP 5)

1. **「ちょうほうけい」の濫用** — `shape_name` Lv2/Lv3 で漢語名「長方形」が4問にわたって出題され、3-6歳は読めない・聞いたこともない可能性が高い (file:116, 123, 129, 131)
2. **「ひしがた」「たまごがた」「ながしかく」の混在** — Lv1 でいきなり「ながしかく」を「rectangle」の正解にしているが、Lv2 では同じ rectangle を「ちょうほうけい」と呼ぶ (file:111 vs 116) — 用語の不統一が混乱を生む
3. **「みぎから N ばんめ」「ひだりから N ばんめ」の高負荷** — `order_color` の半数 (12問) が方向数えで、3-4歳には極めて困難 (file:19, 21, 23, 29, 31, 35, 37, 39, 41, 43, 45, 47, 55, 59, 61, 63, 65)
4. **科学・身体専門用語の早期登場** — Lv2 から「しんぞう」「はい」「ひふ」「じょうはつ」「でんき (雷)」など漢語的概念が連続 (file:155, 167, 172, 192, 197, 393, 397, 405)
5. **答え曖昧問題** — 「りんごは なにいろ？ → あか」(file:276) は緑・黄もあるため幼児が迷う。同じくバナナの黄色 (熟す前は緑) も同様傾向 (file:256)

   - rectangle: 全レベルで「ながしかく」または「しかく」(ちょうほうけい は3歳児向けでは禁句に近い)
   - oval: 「たまごがた」で統一 (既統一 ✓)
   - diamond: 「ひしがた」より「ダイヤのかたち」が直観的

2. **方向数え問題の比率削減**
   - order_color 24問中 12問 (50%) が方向数え。これを 25% (6問) に削減
   - 残りは「色順 (パターン認識)」「色の隣接 (まんなか系)」を増やす

3. **Lv2 の難度調整**
   - 「ちょうほうけい」「しんぞう」「はい」「ひふ」「じょうはつ」「でんき」が Lv2 に集中
   - これらは原則 **Lv3 に格上げ + ヒント必須**

4. **長文選択肢のガイドライン**
   - 1選択肢 8字以内を目安に (現在 15-20字の長選択肢が散見)
   - 因果関係問題 (なぜ系) は Lv3 限定とし、必ず hint フィールド使用

5. **答え曖昧問題の対策**
   - 「りんごは なにいろ？」の類は「ふつうの…」「いちばん おおい…」で限定句を入れる
