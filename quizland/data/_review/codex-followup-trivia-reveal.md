## 3. Q411 ライオンのオス特徴 — 6 個別 PNG (2 シート)

### 質問
- 「ライオンの オスにだけ ある ものは？」
- 正解: たてがみ (index 0)

### スキーマ変更 (重要)
**選択肢を text → image-chip object 配列に変更**:
```js
// 現状: choices: ['たてがみ','しっぽ','つの','はね']
// 新規:
choices: [
  { text:'たてがみ', image:'tategami.png' },
  { text:'しっぽ',   image:'shippo.png' },
  { text:'つの',     image:'tsuno.png' },
  { text:'はね',     image:'hane.png' }
]
```

これにより、 img_word に**メスライオン** (= たてがみなし) を出しても、 子供は「メスかオスか」ではなく「**4 つの体パーツのうちどれがオスにだけある？**」を考えるので、 メスライオンを出すデザインの混乱リスクが減る。

### 生成内容

#### Sheet A: ライオンの reveal stage (1 シート → 2 個別 PNG)
- 横並び 2 パネル、 同じ画風・同じカメラアングル
- Left: `lion_mesu.png` (img_word — メスライオン)
- Right: `lion_osu.png` (img_answer — オスライオン)

##### lion_mesu.png 仕様
- **メスのライオン (たてがみなし)** が画面中央、 サバンナの草原に立つ
- 顔つきは穏やかで子供向け (怒り顔禁止)
- **メスライオンと一目で分かる**ように体格はオスより少し細身、 たてがみ部分はすっきりした顔輪郭

##### lion_osu.png 仕様
- **オスのライオン (たてがみあり)** が同じ構図・同じ画角で立つ
- 顔の周りに**もこもこのたてがみ**が広がる (褐色〜黄褐色、 水彩タッチ)
- 表情は誇らしげだが優しい

#### Sheet B: 4 つの体パーツ image-chip (1 シート → 4 個別 PNG)
- 2×2 グリッドのアセットシート、 各セルに 1 パーツずつ
- 全パネルでライティング・タッチ・サイズ感を共通化
- 背景は**どのパーツも同じ薄い青空 or 透過**

##### 個別ファイル
- `tategami.png` (TL): ライオンの**たてがみ単体** (もこもこ素材、 顔の輪郭なし)、 褐色〜黄褐色
- `shippo.png` (TR): **しっぽ単体** (動物のしっぽ、 細長く先端に房毛)
- `tsuno.png` (BL): **角単体** (シカの角のような枝分かれ角 or 牛の単純な角)、 白〜クリーム色
- `hane.png` (BR): **羽単体** (鳥のはばたく羽、 茶〜白の鳥の羽)

##### 重要
- 各パーツは**単体**で「ライオン本体に取り付く前の素材」感。 **動物本体は一切描かない**
- 4 つを並べて見比べたとき、 「どれが顔の周りに付くだろう？」が直感的に分かる必要なし (それは答え合わせの楽しみ)

### 絶対避けること
- メスライオンを「ただの茶色い犬」「狐」に見えるような描き方 (= ライオンっぽさを失わないように、 体格・耳・しっぽ先端の房はキープ)
- たてがみを過剰にゴージャスにしない (子供が引かないように)

## 4. Q406 バナナの木 — 2 個別 PNG (1 シート)

### 質問
- 「バナナの きは じつは なにの なかま？」
- 選択肢: ['たけ','くさ','まつのき','さくら']
- 正解: 'くさ' (index 1)
- ヒント: 'たかい きに みえるけど…'
- 解説: 'バナナの きは「くさ」の なかまで、ほんとうは「き」じゃないんだよ！'

### 生成内容
**1 アセットシート (横並び 2 パネル)** → 2 個別 PNG:
- `banana_tree_solo.png` (img_word — バナナの木 単体)
- `banana_kusa_compare.png` (img_answer — バナナの木 + 草の仲間並列)

### Left panel: banana_tree_solo.png (img_word)
- **背の高い立派なバナナの木** (画面中央、 大きな葉が複数、 房状に下がるバナナの実)
- 根元は土、 周囲はシンプル (草も生えてない、 何の仲間か分からない構図)
- 「立派な木」と思わせる風格

### Right panel: banana_kusa_compare.png (img_answer)
- **同じバナナの木 (左)** + **草の仲間 (右に並列)**: 例えば**ススキ・シロツメクサ・タンポポ**などの草本植物が 2-3 種
- 高さの対比は維持しつつ、 草とバナナの茎が**同じ繊維っぽい質感**で描かれてる (= 木のような硬い樹皮じゃないことを示唆)
- できれば「草と同じ仲間」を示すなんらかのビジュアル合図 (例: バナナの茎を半透明にして草と同じ繊維パターンが見える、 葉の付け根の構造が同じなど)

### ヒント表現
シートで「バナナは木に見えるけど、 実は草と同じ」という対比が一目で分かる構図を狙う。 ただし強引な指示矢印やラベル文字は避ける (絵だけで語る)。
