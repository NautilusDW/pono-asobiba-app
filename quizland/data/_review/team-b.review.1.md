# Review B1 (independent, difficulty-focused)

## Overall verdict
- **部分採用**: shape_name の構造改善 (line 111 の Lv1 混入修正、Lv1 を 5→8 に拡張) は妥当。ただし Lv1 への `ひしがた` 投入と Lv3 の角数偏重、opposite Lv3 の「なく⇄わらう」「おしえる⇄ならう」の階層に問題あり。

## Difficulty calibration concerns

### shape_name

- **Lv1 #8 `diamond` 「ひしがた」**: 3歳児には語彙として早い。3歳児が知っているのは「まる/しかく/さんかく/ハート/ほし」程度であり、「ひしがた」は園で教わらない。提案者自身も Reviewer note 1 で迷いを認めている。**降格推奨**: 現行どおり Lv2 に残し、Lv1 の枠は別の安全な問題 (例: `heart` を別問題文 "これは なんの かたち？" で重複追加、または `circle` をもう一形式で追加) で埋めるべき。

- **Lv1 #6 `rectangle` 「ながしかく」 / Lv2 #8 `rectangle` 「ちょうほうけい」**: 同一形状を二段階呼称で出す設計は段階学習として理にかなっており、これは**採用可**。ただし「ながしかく」は標準語ではなく方言・俗称寄りなので、子供が知らない可能性がある。`rectangle` Lv1 では「これは どんな かたち？ ／ しかく ／ ながしかく ／ まる ／ さんかく」と「しかく」も正解扱いできる文脈が望ましい (が、4択構造上不可)。妥協として Lv1 では選択肢を「ながしかく / まる / さんかく / ほし」に絞り「しかく」を排除しているのは合理的、**条件付き採用**。

- **Lv1 #7 `oval` 「たまごがた」**: 3歳児にも親しみやすい語であり、選択肢内コントラストも明確。**採用**。

- **Lv2 #1 (line 111 の Lv1 混入修正)**: そのまま Lv2 ブロックへ移動するのは正しい修正。**採用**。

- **Lv3 #3 `triangle` 「なん『かっけい』？」**: これは **6歳でも厳しい**可能性が高い。「さんかっけい」は学校で習う表現であり、未就学児の語彙ではない。提案者は現行のまま維持しているが、選択肢に「ごかっけい/ろっかっけい」が入るとさらに混乱する。**降格または書き換え推奨**: 例えば「このかたちは さんかくの なまえで なんと いう？／さんかっけい・しかっけい・…」のように「正式名は さんかっけい」と教える形に書き換えるか、Lv3 には残すが選択肢を 3 種類に絞る (3/4/5かっけい) のが現実的。

- **Lv3 角数問題の偏重**: 提案者の Reviewer note 2 でも自認されているとおり、Lv3 8問中 5問が角数カウント (`circle 0, star 5, square 4, oval 0, rectangle 4, diamond 4`) になり単調。子供の飽きを招く。**修正推奨**: 削除した「ひしがたは どんな かたちの なかま？」を **書き換えて維持** (例: 「ひしがたを よこに たおすと どんな かたちに みえる？／しかく・…」のような視覚問題)、または「いちばん かどが おおい かたちは どれ？」を 1 問追加。

- **Lv3 #5 `rectangle` 「ちょうほうけいと しかくの ちがいは？」**: 維持に賛成。視覚的に確認できる具体的差異。ただし正答「たてと よこの ながさが ちがう」は文章長め。もう少し短く「たてが ながい」あるいは「たてと よこが ちがう」のほうが 6歳児向き。**微修正推奨**。

### opposite

- **Lv3 #4 「なく⇄わらう」**: ご指摘のとおり**降格すべき**。「なく」「わらう」は 3歳児でも日常使用する具体動作であり、抽象度は Lv1〜Lv2 相当。現行 Lv3 は不適切。**Lv1 または Lv2 へ移動**を推奨。具体的には Lv1 の枠 (例: 現行 Lv1 #4「たつ⇄すわる」と並ぶ動作対) として違和感がない。

- **Lv3 #8 「おしえる⇄ならう」**: 「ならう」は学校的・改まった語彙で 5-6 歳が積極的に使うとは限らない (「ピアノを ならう」程度の受動的接触はある)。古典的に正しい対義ペアではあるが、より子供寄りの「おしえる⇄おそわる」も検討余地あり。ただし「おそわる」も同程度に固いので、**現状維持で許容範囲**。

- **Lv3 #5 「たすける⇄じゃまをする」**: 意味的には正しいが、「じゃまをする」は動詞句で形が長く、他の選択肢「たたかう/あそぶ/やすむ」(全て単動詞) と語形が不揃いで子供にはヒントになりすぎる (一番長い・違和感ある選択肢を選べば当たる)。**選択肢調整推奨**: 全選択肢を「○○する」形に統一する (例: たたかう→けんかする) か、正解を「いじめる」など単動詞に変えるかのどちらか。

- **Lv3 #7 「ほめる⇄しかる」**: 適切。**採用**。

- **Lv2「ふとい⇄ほそい」「あたらしい⇄ふるい」**: 4-5歳で問題なし。**採用**。

- **Lv1 #8 「たくさん⇄すこし」 + ダミー「おおい」**: 「おおい」は意味的に「たくさん」に近すぎる引っかけ問題で、3歳児には酷い。Lv1 (3歳ターゲット) としては難しすぎる。**Lv2 への移動または ダミー差し替え推奨** (「おおい」→「はやい」のみ、「おおきい」のみ等の単純な対比語に)。

## Distractor quality

- **shape_name Lv1 全般**: ダミー選択肢が「まる/しかく/さんかく/ほし/ハート」のローテーションで構成され、絵を見れば消去法で当たる設計。3歳児向けとして妥当。**OK**。

- **shape_name Lv2 #2 `star`**: choices `['まる','しかく','さんかく','ほし']` は Lv1 #5 と完全同一。Lv2 として差別化されておらず、単なる重複。**書き換え推奨** (例: 「ほしの かどは いくつ？」を Lv2 に入れて Lv3 から外し、Lv3 を非角数問題で埋める)。

- **opposite Lv1 #8 「たくさん」**: 上述のとおりダミー「おおい」が紛らわしい。

- **opposite Lv2 #5 「かたい⇄やわらかい」**: ダミーに「やすい」が入っている。「やすい」(安い) は 4-5 歳には意味が捉えづらい上、対比軸 (硬さ→価格) が違うため混乱を招く。**ダミー差し替え推奨** (「つよい」「きびしい」等)。

- **opposite Lv2 #2 「たかい⇄ひくい」**: ダミー「おもい/かたい/くらい」は対比軸が混在して妥当な distractors。**OK**。

- **opposite Lv3 #1 「かるい⇄おもい」**: ダミー「やさしい/ふるい/ちいさい」は対比軸を散らして適切。**OK**。

## Specific question objections

- **Lv1 `diamond` 「ひしがた」**: 3歳児には語彙的に早い。Lv2 維持を推奨。
- **Lv2 `star` 「これは どんな かたち？ ['まる','しかく','さんかく','ほし']」**: Lv1 と完全同形式・同選択肢。Lv2 にする意味がない。差別化必要。
- **Lv3 `triangle` 「なん『かっけい』？」**: 6歳でも語彙的に厳しい。選択肢を「さんかっけい/しかっけい/まる」のように絞るか、Lv2 に降格して「これは さんかくの かたち。せいしきめいは？」など教育的書き方に。
- **Lv3 #4 opposite 「なく⇄わらう」**: 抽象度が Lv1〜2 相当。**Lv1 または Lv2 へ降格必須**。
- **Lv1 opposite 「たくさん⇄すこし」のダミー「おおい」**: 3歳児にはトラップが強すぎる。差し替えまたは Lv2 へ移動を検討。

## Recommended deltas vs implementer's proposal

1. **opposite Lv3 #4 「なく⇄わらう」を Lv2 へ移動** (空いた Lv3 枠は「いそぐ⇄ゆっくり」「ふえる⇄へる」「とおい⇄ちかい」「はじまる⇄おわる」など抽象動作の対比で埋める)。
