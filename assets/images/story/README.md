## 仕様

- 推奨アスペクト比: **16:9 (横長)** または **4:3**
- 推奨最小サイズ: 1280×720
- フォーマット: PNG (または WEBP)
- ファイル名は上記固定 (JS の `OPENING_SCENES` / `ENDING_SCENES` がハードコード参照)
- 画像が存在しない場合は絵文字（🏰👸⚔️ 等）にフォールバックするので、段階的に追加して OK
