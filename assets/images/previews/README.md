# 絵本購入者限定コンテンツのプレビュー画像

`play.html` のプロモモーダル（無料ユーザーがロックカードをタップした時）および
短縮オープニングプロモで表示される、購入者限定ゲームのスクショを置く場所。

詳細な差し替え手順は [docs/SCREENSHOT_GUIDE.md](../../../docs/SCREENSHOT_GUIDE.md) を参照。

## 必要なファイル名

| ゲーム | 枚数 | ファイル名 |
|---|---|---|
| もじかきクエスト | **3** | `writing_1.png` / `writing_2.png` / `writing_3.png` |
| おえかき | 2 | `drawing_1.png` / `drawing_2.png` |
| みちつなぎ | 2 | `slide_1.png` / `slide_2.png` |
| ボウリング | 2 | `bowling_1.png` / `bowling_2.png` |
| ブロックくずし | 2 | `breakout_1.png` / `breakout_2.png` |

合計 **11 枚**。2 枚以上あるゲームはモーダル内で 1.8 秒ごとに自動ローテーション。

## 推奨スペック

- 正方形寄り〜横長（4:3 or 16:9 程度）
- 表示サイズ: 最大 320px 幅 / 36vh 高
- PNG 推奨、300-500KB 以下が目安
- 各ゲームの実プレイ画面 (成功体験のシーン) がベスト。白背景 + キャラの顔見せなど
- 子供が「やってみたい！」と思う瞬間を選ぶ

## 未貼付時の挙動

画像ファイルが存在しない場合は `onerror` で Canvas 生成のプレースホルダー
（カード色 + ゲーム名のダミー画像）に自動フォールバック。
