# 絵本購入者限定コンテンツのプレビュー画像

`play.html` の入場モーダル・おかえりモーダルで 50/50 の確率で表示される、購入者限定コンテンツの紹介スクショを置く場所。

## 必要なファイル名（5種）

| ファイル | 対応ゲーム | カード id |
|---|---|---|
| `writing.png`  | もじかきクエスト | card-writing |
| `drawing.png`  | おえかき         | card-drawing |
| `slide.png`    | みちつなぎ       | card-slide |
| `bowling.png`  | ボウリング       | card-bowling |
| `breakout.png` | ブロックくずし   | card-breakout |

## 推奨スペック

- 正方形寄り〜横長（2:3 〜 4:3 程度）
- 表示サイズ: 最大 260px 幅 / 38vh 高
- PNG または JPG、300-500KB 以下が目安
- 各ゲームのプレイ中ショットや完成画面など、子供が「やってみたい！」と思うシーンがベスト

画像が存在しない場合は `onerror` で自動的に非表示になり、テキスト説明だけ表示されます。
