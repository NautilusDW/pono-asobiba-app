# Quizland リスのくるみちゃん

## キャラクター設定
- **名前**: リスのくるみちゃん（くるみ）
- **種族**: リス（茶色＋クリーム色の毛色）
- **役割**: フクロウ博士のアシスタント。クイズの問題文と選択肢を読み上げる
- **見た目**:
  - ふさふさの大きなしっぽ
  - 首に赤いスカーフ
  - 頭に小さな紅葉/木の実のアクセサリー
  - ぱっちりした大きな目、ピンクの頬、いつも笑顔
- **性格**:
  - 元気で親しみやすい
  - 子供と仲良し、お姉さん感のある優しさ
  - 博士のアシスタントとして頼れる存在

## 採用経緯
- 元々は博士本人がクイズを読む設計だった (owl preset = 年配おじいさん風 babble voice)
- VOICEVOX で博士キャラに合うおじいさん風話者の候補が乏しく、採用断念
- 代わりにアシスタント役の新キャラとしてくるみちゃんを追加
