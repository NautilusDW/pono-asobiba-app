# Codex 画像生成 — 外周ぼかしルール（必須）

## ルール
Codex（または他の画像 AI）に画像生成を頼むとき、**キャンバス全体に絵を描かない**。外周（上下左右）は **意図的にぼかす** ように指示する。

## なぜ
**Why:** 生成された画像は **後で別のフレームに入れ替えたり縮小したりする** ことが頻繁にある。キャンバス目一杯に描き込まれていると、リサイズ・トリミング時に **被写体や重要な要素が外周で切れる** 事故が起きる。外周がぼかしてあれば、切り取っても自然に馴染む。

## 適用判断（How to apply）

### 必ず外周ぼかし指示を出すケース（シーン絵）
- 背景込みのシーン絵: サバンナ + 動物、南の島 + バナナ、雪降る空、教室 + ノート 等
- 風景・情景・状況描写
- 複数要素が画面に配置されている絵

### 例外（外周ぼかし不要のケース）
- **オブジェクト単体・白背景の被写体だけ**: choice 用イラスト（lion.png、teeth.png 等）
- ただし **「上下左右が切れない構図」** は必須条件として伝える（被写体が SAFE エリア内に収まる）

## 発注プロンプトに追記すべき定型文（日本語）
```
- 外周（上下左右）は意図的にぼかし、被写体や背景が画面端で切れないようにする
- 後でリサイズ・トリミングして別フレームに入れ替えるため、外側 10-15% は柔らかく溶け込む処理を必須
- SAFE エリア（内側 80%）に主要被写体を完全に収める
```

## 発注プロンプトに追記すべき定型文（英語）
```
- Soft, blurred edges on all four sides (top/bottom/left/right) — fade outward 10-15%
- Composition will be cropped/resized later for re-framing; ensure no critical elements clipped at edges
- All key subjects must be inside the inner 80% safe zone
```
