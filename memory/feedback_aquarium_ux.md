## 水族館 UX フィードバック

1. **矢印はシンプルに**: 過度な装飾や複雑なUIを避ける
2. **ブーストは大胆に**: 潜水艦ブースト倍率4倍など、変化を感じやすくする
3. **音は柔らかく**: Web Audio合成の音は子ども向けに柔らかいトーンで
4. **複雑さ排除**: おやすみ/成長/性格システムなど複雑な仕組みは不要と判断された
5. **速度キャップ必須**: エサ追尾などで速度が無限蓄積するバグがあった → 常にキャップを入れる

**Why:** 対象ユーザーが5歳児。シンプルで直感的、過剰な複雑さは混乱の元
**How to apply:** 新機能追加時は「5歳児が説明なしで遊べるか」を基準にする。複雑なシステム（育成ステータス等）は提案前にユーザーに確認
