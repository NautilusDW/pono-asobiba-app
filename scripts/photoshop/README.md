# Photoshop スクリプト (キャラクターシート処理)

Pono のキャラクターシート (複数ポーズが 1枚の画像に並んでいる) を
Photoshop で分割・高さ揃えするための ExtendScript (.jsx) 2本。

## 使い方

1. Photoshop でシート画像 (PNG / PSD) を開く
2. **Step 1 — 分割**
   - 分割したいレイヤー (シートそのもの) を選択
   - `File > Scripts > Browse...` → `SplitSheetToLayers.jsx` を実行
   - 行数 / 列数 を入力 (例: 4×3 の 12ポーズ)
   - 生成された "**{元の名前} (split)**" グループ内に 個別レイヤーが並ぶ
3. **Step 2 — 高さ揃え**
   - レイヤーパネルで **対象レイヤーを複数選択** (Ctrl/Cmd + クリック、
     Shift + クリック)。グループを選択した場合は中の全レイヤーが対象。
   - `File > Scripts > Browse...` → `NormalizeHeight.jsx` を実行
   - マーカー方式を使うか選択:
     - **いいえ** (推奨・高速): 画像の外形 (bbox) で揃える
     - **はい** (精密): 各レイヤーの頭のてっぺんに置いた #FF0000 の点を検出
   - 基準高さ (px) を入力 (空欄なら最大高さ)
   - 選択したレイヤーだけが揃う (他のレイヤーは触らない)

## マーカー方式について

ダンスポーズで **手を頭より上に挙げている** 場合、bbox 方式では
手が「トップ扱い」になり キャラクターが相対的に小さくなる問題が出る。

それを避けるには:
1. 分割直後に、各レイヤーの **頭のてっぺん** に 1〜3px の
   真っ赤 (#FF0000) の点 をブラシで入れる
2. `NormalizeHeight.jsx` でマーカー方式を選択

スクリプトが #FF0000 を探して「頭の位置」として使い、手の位置は
無視して揃えてくれる。

## 出力

- すべてのレイヤーが同じ「頭〜足の距離」
- すべての足が **同じ Y 位置 (接地面)** に揃う
- レイヤー別に書き出したい場合は `File > Export > Layers to Files`

## トラブル

- **"グループを選択してください"**: NormalizeHeight は レイヤーセット
  (フォルダアイコンのグループ) を選択してから実行する必要がある
- **スケール後に解像度低下**: 縮小は問題ないが 拡大は画質低下する。
  元シートは十分な高解像度で用意する or スマートオブジェクト化してから実行
- **マーカー方式が非常に遅い**: ExtendScript のピクセル走査は重い。
  レイヤー数が多いと数十秒〜数分かかる。急ぐ時は bbox 方式で
