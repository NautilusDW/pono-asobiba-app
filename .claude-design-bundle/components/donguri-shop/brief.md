# Brand Kit Brief: donguri-shop

> このファイルは Claude Design (Pono LP Brand Kit) 上で並列生成するための仕様書。 生成対象は **ガチャモーダル内 「ショップへ行く」 ボタン + シールショップ画面 UI 装飾の PNG 4 種**。 どんぐり (🌰) を通貨に見立てた温かい木製ショップを描く (栗鼠の貯金的ニュアンス)。 「お金 / 取引」 感を一切ギラギラさせず、 子供 3-6 歳が 「やさしい交換」 と感じる絵にする。 morito が採用案を選定 → Claude Code が `assets/ui/shop/` に保存 → shop UI 実装 → sw.js バンプ → push の流れ。

---

## 用途と文脈 (Pono のあそびば内での役割)

- **配置先 1**: ガチャモーダル内に **「シールショップ」 への入口ボタン** を置く (シールが揃っていない時、 どんぐりで購入できる導線)
- **配置先 2**: シールショップ画面の **背景・値札・購入時パーティクル** の UI 装飾
- **通貨設定**: 「どんぐり (🌰)」 を通貨に見立てる (栗鼠が貯金する感) — 値段表記は別レイヤーで重ねるため画像にはハードコードしない
- **頻度**: ガチャモーダル開いた子供が「シールがダブった / 欲しいのが出ない」 時にショップ導線に流れる。 ショップは 1 セッションで複数回開く可能性あり、 飽きない温かさが必要

---

## 含めるアセット一覧

4 種の PNG を 1 セットで生成する。 すべて **透過 PNG (背景画像のみ不透過可)、 アルファチャンネル必須**。

| ファイル名 | サイズ | 用途 | 透過 |
| --- | --- | --- | --- |
| `shop_entry_button.png` | 280×80 (1x) / 560×160 (@2x) | ガチャモーダル内の入口ボタン (木製看板風) | 透過 |
| `shop_background.png` | 1080×1920 (縦長 PWA) | ショップ画面の背景 (木材店内 + シール瓶の棚) | 不透過 可 |
| `shop_price_tag.png` | 120×60 (1x) / 240×120 (@2x) | 値札 (木+紐の手作り感、 数字は別レイヤー) | 透過 |
| `shop_purchase_celebration.png` | 512×512 | 購入完了時の「やったー！」 パーティクル (中央寄せ) | 透過 |

- 背景以外は **透過 PNG** (アルファチャンネル必須)
- 背景は PNG 不透過でも JPEG でも可だが、 アプリ統一のため PNG 推奨
- 背景はタイル化可能なら範囲縮小可 (例: 540×960 を縦タイルで使用) — その場合は別途 brief 追記

## アートディレクション (色・トーン・既存 Brand Kit との整合)

### 既存 Brand Kit 一貫性 (必読)

- **基調**: Pono LP / play.html 既存の **温かい木調 + 紙 + 毛糸** のクラフト感
- **店内テーマ**: 「森の中の小さな木製ショップ」 — 栗鼠の貯金 (どんぐり) で買い物をする世界観
- **色**:
  - 木材ベース: 温かい木目 (`#C8A074` 〜 `#9E7B4A` 系、 真鍮 NG)
  - シール瓶: クリアガラス + 中身は淡パステル (シール群を瓶詰めで陳列)
  - 値札の紐: 焦茶 (`#5D4E37` 系、 hero-labels と同調)
  - パーティクル: pastel kawaii (淡赤 `#F7B7A3` / 淡緑 `#C8D9A0` / 淡黄 `#F7DCA0` / 淡水 `#A8D0D9`)
- **質感**:
  - 水彩 + 鉛筆縁取りで手描き感を出す (hero-labels と同等)
  - 紙繊維 / 木目テクスチャ は控えめに (背景は少し強め、 ボタン/値札は控えめ)
- **「お金 / 取引」 をギラギラさせない**:
  - メタリック金 / 真鍮 / クロム は **禁止**
  - $ / 円 / 通貨マーク は **禁止** (どんぐり🌰のみ)
  - パチンコ / ソシャゲ / ガチャゲー感は **禁止**
  - 「温かい交換」 「栗鼠が木の実を交換してくれる」 のニュアンスを死守

### 子供 3-6 歳ターゲットの直感性

- 「木のお店 = 安全な場所」 を直感
- ボタンは押せる感が必要 (木製看板の立体感、 影は控えめ)
- ショップ背景は **賑やかすぎない** (シール瓶 5-7 個程度の棚、 中央に主役シール 1-2 個、 子供が選びやすい余白)

## 配置先 / 利用シーン

### shop_entry_button.png
- **配置先**: ガチャモーダル内の下部
- **表示サイズ**: 280×80 (1x)、 Retina で 560×160
- **クリック動作**: シールショップ画面へ遷移
- **PNG 内テキスト**: 「シールショップ」 (ひらがな+カタカナ、 焦茶手書き調、 hero-labels と同フォント感) — 看板に焼き込み。 🛍️ 等の絵文字は使わず、 必要なら小さな手描きアイコン (シール束 or 木の棚) を看板内に配置可

### shop_background.png
- **配置先**: シールショップ画面 (`shop.html` or modal) の背景
- **表示サイズ**: 1080×1920 (縦長 PWA 想定) — タイル化可能なら範囲縮小可
- **シーン構成**:
  - 上部: 木材の壁面 + シール瓶が並んだ棚 (3-5 個の瓶)
  - 中央: 商品陳列領域 (UI 側でシール一覧を重ねるため、 ここは余白寄り、 木の床 or テーブル面)
  - 下部: 木の床 + 小さなどんぐり 2-3 個のアクセント
- **UI 重ね対応**: 中央 60% は UI 重ねるため装飾を最小限に (タイトル / シール一覧 / 値札の重ね先)

### shop_price_tag.png
- **配置先**: シール 1 個ごとの値札
- **表示サイズ**: 120×60 (1x)、 Retina で 240×120
- **値段表記**: 「50 🌰」 等は **画像にハードコードしない** (CSS/HTML で別レイヤーとして数字+🌰を重ねる)
- **画像内容**: 木製の小札 + 紐 (棚から吊るす感) のみ。 中央は空白で値段を重ねやすい構図

### shop_purchase_celebration.png
- **配置先**: 購入完了時に中央に 0.8 秒フェードイン
- **表示サイズ**: 512×512 (中央寄せ、 1x のみ可)
- **演出内容**: 紙吹雪 + 小さな🌰がはじける + 「やったー！」 の手描き文字 (ひらがな、 焦茶) — 全体に透明感あり、 UI の購入ボタンを隠さない

---

## 禁止事項 (絶対やらない)

- **SVG / Canvas / CSS / PIL / 手書きベクター での代用は禁止** — PNG ラスター生成のみ
- **MCP (higgsfield / fal-ai) で Claude Code が直接生成するのは禁止** — 必ず Claude Design (Pono LP Brand Kit) 経由
- **emoji 直接埋め込み (🛍️ / 🌰 / 💰) を画像化しただけは禁止** — 手描き水彩+鉛筆縁取りで描き起こす
- **メタリック金 / 真鍮 / クロム / コイン表現は禁止** — 木とどんぐりで代替
- **$ / 円 / 通貨マーク / お札 / クレジットカード絵柄は禁止**
- **パチンコ / ソシャゲ / ガチャゲー風のキラギラ演出は禁止**
- **値段数字を画像にハードコードしない** — 数字は別レイヤー
- **キャラクター (ポノ / ハリネズミ / 顔表情) の描画は禁止** — 装飾は木とどんぐりとシール瓶のみ
- **背景色付きで透過 PNG にしない (透過必須のアセット)** — `shop_entry_button` `shop_price_tag` `shop_purchase_celebration` は透過必須
- **賑やかすぎる背景** — 子供の選択を邪魔しない余白を中央に確保
