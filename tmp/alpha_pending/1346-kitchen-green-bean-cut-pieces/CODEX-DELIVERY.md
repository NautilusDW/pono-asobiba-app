# 1346 kitchen green bean cut pieces

- `green_bean_cut_pieces_raw.png`: GPT Image 2 raw sheet, generated from the current whole green-bean asset and the old cropped-looking piece.
- `green_bean_cut_pieces_alpha.png`: local chroma-key removal result.
- `process_green_bean_sheet.py`: splits the 2 x 5 sheet, trims transparent padding, and writes ten runtime sprites.

The ten pieces have visible knife-cut cross-sections, varied curvature and viewpoint, and no plate, text, or overlapping objects. The raw file remains unchanged in this delivery folder. Runtime outputs replace `assets/images/bento/cooking/ninjin_ingen/pieces/green_bean_piece_01.png` through `_10.png`.
