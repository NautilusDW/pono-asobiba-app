## 修正方針

VOICEVOX は「ひらがな羅列＋スペース区切り」よりも「漢字混じり＋スペース無しの自然文」の方が単語境界推定の精度が高く、誤読が減る。以下の方針で `speech` 列のみを修正する (`display` 列は子供向け表示用で絶対に変更しない)。

1. **スペース除去**: 分かち書きスペースをすべて削除し、自然な日本語文にする
   - 例: `まんなかは なにいろ？` → `真ん中は何色？`
2. **漢字化 (一般読みの名詞・動詞・形容詞)**: 助詞は平仮名のまま、一般的読みの内容語は漢字化
   - 例: `あそびに きたよ` → `遊びに来たよ`
3. **助詞・助動詞は平仮名**: 「は」「を」「が」「の」「に」「と」など
4. **子供向けの優しさは維持**:
   - カタカナ語 (ピンク/オレンジ/ハート 等) はカタカナのまま
   - 過度な専門用語は使わない
   - 誤読しやすい / 子供向けに固い漢字 (例: 「林檎」「象」) は平仮名のまま残す
5. **数字は半角アラビアで統一** (VOICEVOX が文脈で「に」「ふたつ」を読み分ける部分は user dict / accent_overrides 側で吸収)
6. **カギ括弧 (Q95)**: 既存ルールに合わせて除去 (画面表示側のみカギ括弧を残す)
7. **同音異義 (Q160 肺/胃)**: 漢字化で第一段の解消を狙い、保険として `_alt` 補足版を残す

## 漢字化の方針まとめ

### 漢字化したもの (一般読み・誤読リスク低)
- 名詞: 真ん中 / 何色 / 反対 / 息 / 体 / 肺 / 胃 / 心 / 指 / 臓器 / 胃袋 / 次
- 動詞: 吸う / 来た / 遊ぶ
- 形容詞: 大きい / 小さい / 赤い / 丸い / 優しい

### 平仮名のまま残したもの
- 子供向けに固い / 誤読しやすい: りんご (林檎は固い), あか / あお / きいろ / みどり (色の単独発話は平仮名で安定)
- 助詞: は / を / が / の / に / と / で
- カタカナ語: ピンク / オレンジ など (本テスト 27 ファイルには非該当)
- 和語数詞: ふたつ / みっつ / よっつ / いつつ (辞書登録済 + 漢字化すると逆に誤読)

## 想定される効果

- 「真ん中わ」誤読の解消 (「真ん中は」と漢字化することで助詞「は」が明確化)
- 「あそびに きたよ」のような分かち書きスペースに起因する不自然な切れ目の解消
- 「肺」「胃」の漢字化で「返事のはい」「助詞のい」との取り違えリスクが減少
- 「大きいの反対は？」で形容詞の終止形「い」と助詞「の」の境界が明確化
- 全体として VOICEVOX の単語境界推定精度向上 → 抑揚・アクセントが自然化
