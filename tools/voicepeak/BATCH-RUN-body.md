## body 特有の試聴ポイント

> **今回も漢字混じり化**: 下表の例文・例語は CSV と同じ漢字混じり版で記載。 VOICEPEAK の形態素解析・アクセント推定の精度を上げる目的。 助詞 「は / の / を / が / と」 はカナ維持。
>
> **ただし誤読リスクが高い箇所はカナ維持** (ユーザー方針 2026-05-17)。 body Phase 1 では以下 10 箇所を事前にカナ戻し済 — **試聴駆動から事前防止へシフト** (shape_name / weather / opposite と同方針):
> - 「目 → め」 (1 モーラ短語、 「目 (もく)」 揺れリスク。 ペア相手「ま (眼)」 連想を除き、 子供向け文脈で 1 モーラ漢字は読み崩れ確率最大)
> - 「手 → て」 (1 モーラ短語、 「手 (しゅ)」 揺れリスク。 慣用句「手」 だけだと「シュ」 と読まれる事例あり)
> - 「足 → あし」 (足 / 脚 / 葦 / 芦 同形異義語、 「足 (そく)」 揺れ + 「足し算」 連想干渉。 体パーツ文脈確定でも 2 モーラ短語ゆえ揺れ)
> - 「花 / 鼻 / 端 → はな」 (花 / 鼻 / 端 同形異義語、 子供向け文脈では「鼻」 だが「花」 と混同しやすい)
> - 「歯 / 葉 / 端 / 刃 → は」 (1 モーラ + 助詞「は」 と同形 + 「歯/葉/端/刃」 同形異義語多数。 体パーツ文脈確定でも音的に助詞と区別困難)
> - 「舌 / 下 → した」 (舌 (シタ) / 下 (シタ) 同形異義 + 動詞過去形「〜した」 と同形。 文脈で「舌」 と確定できても VOICEPEAK のアクセント推定で揺れる)
> - 「肺 / 灰 / 杯 / 拝 / 牌 → はい」 (同形異義語多数 + 返事「はい」 と同形。 子供向け文脈で「肺」 (ハ↓イ) と読ませたいが、 短語ゆえ揺れリスク最大。 辞書登録済 (L61: 肺 / ハイ / 0) だが、 CSV ではカナ維持で安全側)
> - 「生える / 映える / 栄える → はえる」 (生える (はえる、 核 2) / 映える / 栄える 同形異義語、 「歯が はえる」 文脈で「生える」 と確定したいが揺れる)
> - 「治す / 直す → なおす」 (治す (傷を治す) / 直す (壊れたものを直す) 同形異義、 「傷を なおす」 文脈で「治す」 と確定したいが揺れる)
> - 常用外漢字 6 語: 「怪我 → けが」 「瘡蓋 → かさぶた」 「瞬き → まばたき」 「模様 → もよう」 「震える / 振るえる → ふるえる」 「御腹 / お腹 → おなか」 (いずれも常用漢字外 or 子供向けにはひらがな表記が一般的)
>
> 漢字のまま残した語 (= 99% 確実に正しく読まれると判断、 試聴で確認): 体 / 口 / 耳 / 片手 / 指 / 何本 / 5本 / 何 / 体全体 / 包む / 守る / 皮膚 / 血 / 体じゅう / 送る / 心臓 / 食べ物 / 噛む / 息 / 吸う / 臓器 / 眠る / 休む / 抜ける / 新しい / 赤ちゃん / 骨 / 大人 / 多い / 少ない / 寒い / 温める / 中 / 空気 / 動く / 先 / 指紋 / 傷。

### 結論

**body Phase 1 は名詞 (「皮膚 / 心臓 / 指紋 / 血 / 骨 / 大人 / 赤ちゃん / 片手 / 何本 / 5本 / 何 / 体全体 / 体じゅう / 食べ物 / こなごな / 中 / 空気 / 先 / 傷」)、 動詞 (「包む / 守る / 噛む / 眠る / 休む / 抜ける / 温める / 動く」)、 形容詞 (「新しい / 多い / 少ない / 寒い」) の 30+ 語が全て未登録の可能性が高い**。 とくに 「皮膚 / 心臓 / 指紋」 の医学用語、 「血」 (1 モーラ + 同形異義語)、 「噛む」 (常用漢字外)、 「赤ちゃん / 大人 / 骨」 (音読み混同リスク)、 「温める / 新しい」 (5 モーラ長語) が読み崩れリスクの要注意ポイント。 試聴で読み崩れが顕著であれば、 別タスクで上記候補を辞書追加 → .vdc2 再生成 → 再書き出しのループを回す。
