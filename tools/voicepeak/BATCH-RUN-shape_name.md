## shape_name 特有の試聴ポイント

> **今回から漢字混じり化**: 下表の例文・例語は CSV と同じ漢字混じり版で記載。 VOICEPEAK の形態素解析・アクセント推定の精度を上げる目的。 「どんな / どれ / どちら」 はカナ維持。 「ハート / ダイヤ / ピザ / ノート / サッカーボール / ゼロ」 はカタカナ維持。

| ポイント | 確認内容 |
|---|---|
| 形名アクセント (登録済) | 「丸」 (マル, 核 1)、 「四角」 (シカク, 核 0 = 平板)、 「三角」 (サンカク, 核 0 = 平板)、 「星」 (ホシ, 核 0 = 平板) が辞書通りに読まれるか。 とくに 「四角」 が 「シ↓カク」 (頭高) でなく 「シ↑カク↑」 (平板) になっているか |
| 「ハート」 (カタカナ) | 「ハート」 (核 1 想定: ハ↓ート) が外来語として自然に読まれるか。 「ハ ↑ ート↑」 平板化していないか |
| 「卵形」 アクセント | 「卵形」 (タマゴガタ, 5 モーラ) が 「タマ ↑ ゴガタ↑」 と自然に流れるか。 「タマ ↓ ゴ ↓ ガタ」 と途中で核が入っていないか。 漢字混じり化により VOICEPEAK が 「タマゴガタ」 と一語で認識するか確認。 (※ 未登録 — 試聴で要確認) |
| 「菱形」 アクセント | 「菱形」 (ヒシガタ, 4 モーラ) が中高 (ヒ ↑ シガタ ↓) か平板 (ヒ ↑ シガタ↑) で素直に読まれるか。 「菱形」 + 「の」 助詞接続が自然か。 「菱」 は常用外漢字なので VOICEPEAK が 「ヒシガタ」 と読めているか要注意。 (※ 未登録 — 試聴で要確認) |
| 「ダイヤの 形」 連結 | 「ダイヤ」 (核 1 想定: ダ↓イヤ) + 「の」 + 「形」 (カタチ) が自然に連結されるか。 「ダ ↓ イヤ ↓ ノ ↓ カタ ↓ チ」 とブツ切れになっていないか |
| 「葉っぱ」 促音 | 「葉っぱ」 (ハッパ, 核 1: ハ↓ッパ) の促音が自然に詰まって読まれるか。 「ハ ↑ ッパ↑」 と平板化していないか。 漢字 + 「っぱ」 混在表記で VOICEPEAK が一語として認識するか確認 |
| 「○個」 単位 (五個 / 三個 / 四個 / ゼロ個) | 「五個」 (ゴ↑コ↓ or ゴ↓コ?)、 「三個」 (サ↓ンコ or サンコ↑?)、 「四個」 (ヨ↑ンコ↓ or ヨ↓ンコ?)、 「ゼロ個」 (ゼ↓ロコ?) が単一語として自然に読まれるか。 「五 + 個」 「三 + 個」 と切れた読みになっていないか (= 数詞 + 助数詞接続の確認、 漢字化により VOICEPEAK が数詞 + 助数詞として正しく解析するか要試聴) |
| 疑問詞「どんな」 | 「これは どんな 形？」 の 「ど ↑ んな↑」 が自然な疑問詞として読まれるか。 「ど ↓ ンナ」 (頭高) になっていないか。 文末 「カタ ↑ チ↑?」 が疑問の上昇調か |
| 疑問詞「何の」 | 「これは 何の 形？」 の 「何の」 (ナ↓ンノ or ナンノ↑?) が自然に読まれるか。 「何の + 形」 の連結に違和感がないか。 漢字 「何」 が 「ナン」 と読まれるか (「ナニ」 と誤読しないか) 要確認 |
| 疑問詞「どれ」 | 「真ん丸い ピザを ちょうど 半分に 切ると どれ？」 の 「ど ↑ れ↑?」 が疑問の上昇調になるか。 文全体が 22 モーラと長いので途中で息継ぎが不自然に入っていないか |
| 「角」「形」 接続 | 「星の 角は 何個？」 「この形の 角は 全部で 幾つ？」 等で 「角」 (カド, 核 1)、 「形」 (カタチ, 核 0 想定) の助詞接続が自然か。 ※ 「角」 が 「カク」 と誤読される可能性に最大注意 (カド/カク 揺れ語、 辞書登録があれば優先される)。 (※ どちらも未登録の場合は誤読リスク高) |
| 「全部で 幾つ」 連結 | 「全部で 幾つ」 (ゼンブデ + イクツ) の連結が自然か。 「ゼンブ ↓ デ ↓ イ ↓ クツ」 とブツ切れになっていないか |
| カタカナ語 (ピザ / ノート / サッカーボール) | 「ピザ」 (ピ↓ザ, 核 1) / 「ノート」 (ノ↓ート, 核 1) / 「サッカーボール」 (サッカーボール, 6 モーラ) が外来語として自然に読まれるか。 「サッカーボール」 が 1 語として流れるか、 「サッカー + ボール」 と分かれて読まれていないか |
| 「真ん丸い」 形容詞 | 「真ん丸い ピザ」 の 「真ん丸い」 (マンマルイ, 核 3 想定: マンマ ↓ ルイ) が自然な形容詞として読まれるか。 「ピザ」 への接続が自然か。 漢字 + ひらがな混在表記で VOICEPEAK が形容詞として認識するか確認 |
| 「どちらが」「多い」 | 「星と 丸では どちらが 角が 多い？」 「丸と 三角では どちらが 角が 多い？」 で 「どちらが」 (ドチラガ, 核 1 想定: ド ↓ チラガ?) と 「多い」 (オーイ, 核 2 想定: オ↑ーイ ↓) が自然な比較疑問として読まれるか。 漢字 「多い」 が 「オオイ」 (オーイ) と読まれるか (「タイ」 と誤読しないか) 要確認 |
| 「切ると」 動詞 | 「真ん丸い ピザを ちょうど 半分に 切ると どれ？」 の 「切ると」 (キルト, 核 1 想定) が自然に読まれるか。 「半分に」 (ハンブンニ) + 「切ると」 の連結が自然か |
