## shape_name 特有の試聴ポイント

| ポイント | 確認内容 |
|---|---|
| 形名アクセント (登録済) | 「まる」 (マル, 核 1)、 「しかく」 (シカク, 核 0 = 平板)、 「さんかく」 (サンカク, 核 0 = 平板)、 「ほし」 (ホシ, 核 0 = 平板) が辞書通りに読まれるか。 とくに 「しかく」 が 「シ↓カク」 (頭高) でなく 「シ↑カク↑」 (平板) になっているか |
| 「ハート」 (カタカナ) | 「ハート」 (核 1 想定: ハ↓ート) が外来語として自然に読まれるか。 「ハ ↑ ート↑」 平板化していないか |
| 「たまごがた」 アクセント | 「たまごがた」 (タマゴガタ, 5 モーラ) が 「タマ ↑ ゴガタ↑」 と自然に流れるか。 「タマ ↓ ゴ ↓ ガタ」 と途中で核が入っていないか。 (※ 未登録 — 試聴で要確認) |
| 「ひしがた」 アクセント | 「ひしがた」 (ヒシガタ, 4 モーラ) が中高 (ヒ ↑ シガタ ↓) か平板 (ヒ ↑ シガタ↑) で素直に読まれるか。 「ひしがた」 + 「の」 助詞接続が自然か。 (※ 未登録 — 試聴で要確認) |
| 「ダイヤの かたち」 連結 | 「ダイヤ」 (核 1 想定: ダ↓イヤ) + 「の」 + 「かたち」 が自然に連結されるか。 「ダ ↓ イヤ ↓ ノ ↓ カタ ↓ チ」 とブツ切れになっていないか |
| 「はっぱ」 促音 | 「はっぱ」 (ハッパ, 核 1: ハ↓ッパ) の促音が自然に詰まって読まれるか。 「ハ ↑ ッパ↑」 と平板化していないか |
| 「○こ」 単位 (ごこ / さんこ / よんこ / ぜろこ) | 「ごこ」 (ゴ↑コ↓ or ゴ↓コ?)、 「さんこ」 (サ↓ンコ or サンコ↑?)、 「よんこ」 (ヨ↑ンコ↓ or ヨ↓ンコ?)、 「ぜろこ」 (ゼ↓ロコ?) が単一語として自然に読まれるか。 「ご + こ」 「さん + こ」 と切れた読みになっていないか (= 数詞 + 助数詞接続の確認、 未登録なので試聴必須) |
| 疑問詞「どんな」 | 「これは どんな かたち？」 の 「ど ↑ んな↑」 が自然な疑問詞として読まれるか。 「ど ↓ ンナ」 (頭高) になっていないか。 文末 「カタ ↑ チ↑?」 が疑問の上昇調か |
| 疑問詞「なんの」 | 「これは なんの かたち？」 の 「なんの」 (ナ↓ンノ or ナンノ↑?) が自然に読まれるか。 「なんの + かたち」 の連結に違和感がないか |
| 疑問詞「どれ」 | 「まんまるい ピザを ちょうど はんぶんに きると どれ？」 の 「ど ↑ れ↑?」 が疑問の上昇調になるか。 文全体が 22 モーラと長いので途中で息継ぎが不自然に入っていないか |
| 「かど」「かたち」 接続 | 「ほしの かどは なんこ？」 「このかたちの かどは ぜんぶで いくつ？」 等で 「かど」 (カド, 核 1)、 「かたち」 (カタチ, 核 0 想定) の助詞接続が自然か。 (※ どちらも未登録) |
| 「ぜんぶで いくつ」 連結 | 「ぜんぶで いくつ」 (ゼンブデ + イクツ) の連結が自然か。 「ゼンブ ↓ デ ↓ イ ↓ クツ」 とブツ切れになっていないか |
| カタカナ語 (ピザ / ノート / サッカーボール) | 「ピザ」 (ピ↓ザ, 核 1) / 「ノート」 (ノ↓ート, 核 1) / 「サッカーボール」 (サッカーボール, 6 モーラ) が外来語として自然に読まれるか。 「サッカーボール」 が 1 語として流れるか、 「サッカー + ボール」 と分かれて読まれていないか |
| 「まんまるい」 形容詞 | 「まんまるい ピザ」 の 「まんまるい」 (マンマルイ, 核 3 想定: マンマ ↓ ルイ) が自然な形容詞として読まれるか。 「ピザ」 への接続が自然か |
| 「どちらが」「おおい」 | 「ほしと まるでは どちらが かどが おおい？」 「まると さんかくでは どちらが かどが おおい？」 で 「どちらが」 (ドチラガ, 核 1 想定: ド ↓ チラガ?) と 「おおい」 (オーイ, 核 2 想定: オ↑ーイ ↓) が自然な比較疑問として読まれるか |
