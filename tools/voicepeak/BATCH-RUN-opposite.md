## opposite 特有の試聴ポイント

> **今回から漢字混じり化**: 下表の例文・例語は CSV と同じ漢字混じり版で記載。 VOICEPEAK の形態素解析・アクセント推定の精度を上げる目的。 助詞 「は / の / を / が」 はカナ維持。
>
> **ただし誤読リスクが高い箇所はカナ維持** (ユーザー方針 2026-05-17 = 「漢字化するときに間違えそうなところは、漢字じゃなくてもいいからね」)。 opposite Phase 1 では以下 11 箇所を事前にカナ戻し済 — **試聴駆動から事前防止へシフト** (shape_name / weather と同方針):
> - 「冷たい」 → 「つめたい」 (ツメタイ / レイタイ揺れ、 短い形容詞での誤読回避)
> - 「後ろ」 → 「うしろ」 (送り仮名 「後」「後ろ」 揺れ + 「後 (ご / あと / うしろ)」 多義読み回避)
> - 「立つ」 → 「たつ」 (立つ / 断つ / 絶つ / 経つ / 起つ / 建つ 同形異義語多数)
> - 「開ける」 → 「あける」 (開ける / 明ける / 空ける 同形異義語、 文脈で曖昧)
> - 「閉める」 → 「しめる」 (閉める / 締める / 占める / 湿る / 絞める 同形異義語多数)
> - 「沢山」 → 「たくさん」 (「沢」「山」 個別読み崩れリスク + 常用外複合)
> - 「早い / 速い」 → 「はやい」 (同形異義語、 「速度」 と 「時刻」 の文脈区別困難)
> - 「固い / 堅い / 硬い」 → 「かたい」 (三つの同形異義語、 物理的硬度文脈だが揺れ)
> - 「泣く / 鳴く / 無く」 → 「なく」 (同形異義語、 人間文脈だが揺れリスク)
> - 「暗い」 → 「くらい」 (暗い / 位 / 〜ぐらい (副助詞) 同形異義語、 短 2 モーラなので誤読時の影響大)
> - 「帰る / 返る / 変える / 換える / 蛙」 → 「かえる」 (同形異義語多数、 動詞 + 名詞 (蛙) まで含む)

### 結論

**opposite Phase 1 は形容詞 (「大きい / 小さい / 暑い / 高い / 低い / 長い / 短い / 太い / 細い / 強い / 弱い / 多い / 少ない / 軽い / 重い / 明るい / 新しい / 古い / 遅い / 柔らかい」)、 動詞 (「座る / 笑う / 教える / 習う / 助ける / 出かける / 褒める / 叱る」)、 名詞 (「夜 / 朝 / 邪魔 / 好き / 嫌い / 少し」) が全て未登録の可能性が高い**。 とくに 「暑い ↔ 熱い ↔ 厚い」 「重い ↔ 思い」 「古い ↔ 振るい」 「好き ↔ 隙」 「夜 ↔ 寄る」 等の同形異義語、 および 「柔らかい / 新しい / 助ける / 教える / 出かける / 少ない / 明るい / 短い / たくさん」 等の 4-5 モーラ語が読み崩れリスクの要注意ポイント。 試聴で読み崩れが顕著であれば、 別タスクで上記候補を辞書追加 → .vdc2 再生成 → 再書き出しのループを回す。
> **なお最高リスク 11 箇所 (「冷たい / 後ろ / 立つ / 開ける / 閉める / 沢山 / 早い / 固い / 泣く / 暗い / 帰る」) は 2026-05-17 方針で CSV 段階でカナ戻し済 = 試聴駆動から事前防止へシフト**。 残るリスクは漢字維持グループ (同形異義語・5 モーラ長語) のみで、 試聴の負担は大幅に低減。
