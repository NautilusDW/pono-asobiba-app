```powershell
cd d:\AppDevelopment\pono-asobiba-app

  -InputDir   "D:\AppDevelopment\pono-asobiba-app\tmp\quizland_NA\trivia" `

  -OutputDir  "D:\AppDevelopment\pono-asobiba-app\tmp\quizland_NA\trivia_expanded"
```

## trivia 特有の試聴ポイント

> **今回も漢字混じり化**: 下表の例文・例語は CSV と同じ漢字混じり版で記載。 VOICEPEAK の形態素解析・アクセント推定の精度を上げる目的。 助詞 「は / の / を / が / と / で / に / にだけ」 はカナ維持。
