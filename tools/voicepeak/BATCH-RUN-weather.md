## Step 3: Expand スクリプト実行 (q番号への展開)

```powershell
cd d:\AppDevelopment\pono-asobiba-app
powershell -ExecutionPolicy Bypass -File tools\voicepeak\Expand-VoicepeakUniqueWavs.ps1 `
  -InputDir   "D:\AppDevelopment\pono-asobiba-app\tmp\quizland_NA\weather" `
  -UniqueCsv  "tools\voicepeak\voicepeak_lines_unique_phase1_weather.csv" `
  -ExpandJson "tools\voicepeak\voicepeak_unique_expand_weather_phase1.json" `
  -OutputDir  "D:\AppDevelopment\pono-asobiba-app\tmp\quizland_NA\weather_expanded"
```

→ `weather_expanded/` 配下に q###_q.wav / q###_a.wav (or _b/_c/_d) (合計 48 件、 q110 〜 q133 の Phase 1 全展開) が生成される。

## weather 特有の試聴ポイント

> **今回から漢字混じり化**: 下表の例文・例語は CSV と同じ漢字混じり版で記載。 VOICEPEAK の形態素解析・アクセント推定の精度を上げる目的。 「どんな / どこから / どうして / なぜ / いつ / なんという」 はカナ維持。 助詞・接続助詞もカナ維持。

| ポイント | 確認内容 |
|---|---|
| 天気名アクセント (短語) | 「晴れ」 (ハレ, 核 0 想定: 平板)、 「雨」 (アメ, 核 1: ア↓メ)、 「曇り」 (クモリ, 核 3 = 尾高想定)、 「雪」 (ユキ, 核 2 = 尾高)、 「虹」 (ニジ, 核 2 = 尾高)、 「雷」 (カミナリ, 核 3)、 「風」 (カゼ, 核 0 = 平板)、 「雲」 (クモ, 核 1: ク↓モ) が自然に読まれるか。 とくに 「雨」 (ア↓メ) と 「飴」 (アメ↑ = 平板) の混同に注意 — 「飴」 と誤って平板で読むと意味不明になる |
| 「天気」 アクセント | 「天気」 (テンキ, 核 1 想定: テ↓ンキ) が自然な疑問の冒頭として読まれるか。 「この 天気は？」 で 「テ↑ンキ」 平板化していないか。 漢字 「天気」 が一語として認識されているか |
| 「竜巻 / 台風」 4-モーラ複合 | 「竜巻」 (タツマキ, 核 0 想定: 平板 タ↑ツマキ↑)、 「台風」 (タイフー, 核 0 = 平板 想定) が自然な複合語として読まれるか。 「タ ↓ ツマキ」 (頭高) や 「タ ↓ イフー」 (頭高) になっていないか |
| 「七色」 (ななしょく) | 「七色」 (ナナショク, 核 0 = 平板 想定) が自然に読まれるか。 漢字混じり化で 「シチショク」 や 「ナナイロ」 と誤読されないか要注意 — 期待は 「ナナショク」。 (※ 未登録 — 漢字読み揺れリスク大) |
| 「雪国」 (ゆきぐに) | 「雪国」 (ユキグニ, 核 2 想定: ユキ↑グニ↓ の尾高 or ユキ↓グニ 中高) が自然に読まれるか。 「ユキ + クニ」 と切れていないか。 漢字 「国」 が 「クニ」 と濁って 「グニ」 になっているか (連濁) |
| 「温度計」 (おんどけい) | 「温度計」 (オンドケー, 核 3 想定: オンド↑ケー↓) が自然に読まれるか。 「オン + ド + ケイ」 と切れていないか。 漢字 「計」 が 「ケー」 (長音) として自然か |
| 「蒸発」 (じょうはつ) | 「蒸発」 (ジョーハツ, 核 0 = 平板 想定) が自然に読まれるか。 「ジョー + ハツ」 と切れていないか。 「蒸」 の長音 「ジョー」 が自然に伸びるか |
| 「結晶」 (けっしょう) | 「雪の 結晶は どんな 形？」 の 「結晶」 (ケッショー, 核 0 = 平板 想定) が自然に読まれるか。 「ケッ + ショー」 と促音 + 長音が崩れていないか。 「結晶」 + 「は」 助詞の連結が自然か |
| 「水溜まり」 (みずたまり) | 「水溜まり」 (ミズタマリ, 核 0 = 平板 想定) が自然に読まれるか。 漢字混じり化により VOICEPEAK が 「ミズタマリ」 と一語認識するか確認。 「ミズ + タマリ」 と切れていないか。 (※ 未登録) |
| 動詞: 「降る / 落ちる / 鳴る / 出る / 上る / 冷える / 見える」 | 各動詞 (フ↓ル / オ↓チル / ナ↓ル / デ↑ル↓ / ノ↑ボル↓ / ヒ↑エル↓ / ミ↑エル↓) が辞書通りに読まれるか。 漢字混じり化により 「降る」 → 「フル」、 「落ちる」 → 「オチル」 と正しく読まれるか。 「降る → コーフル」 「落ちる → ラクチル」 等の誤読がないか |
| 形容詞: 「白い / 暑い / 高い / 涼しい」 | 「白い」 (シロ↑イ↓, 核 2)、 「暑い」 (ア↑ツイ↓, 核 2)、 「高い」 (タカ↑イ↓, 核 2)、 「涼しい」 (スズ↑シイ↓, 核 3 想定) が自然な形容詞として読まれるか。 「暑い」 と 「熱い」 (どちらもアツイ) の同形異義語 — 「夏は 暑い」 文脈で問題なく読まれるか |
| 「靄」 (もや) | 「朝、白い 靄が 出る 天気を 何という？」 の 「靄」 (モヤ, 核 1 想定: モ↓ヤ) が自然に読まれるか。 「靄」 は常用外漢字なので VOICEPEAK が 「モヤ」 と読めているか要注意 — 読めない場合はひらがな表記に戻すか辞書登録を検討。 (※ 未登録 — 読み崩れリスク高) |
| 「粒」 (つぶ) | 「息の 中の 水が 冷えて 粒に なるから」 の 「粒」 (ツブ, 核 1 想定: ツ↓ブ) が自然に読まれるか。 「粒」 + 「に」 助詞の連結が自然か。 (※ 未登録) |
| 「1年」 (いちねん) 数詞 + 助数詞 | 「1年で 一番 暑い 季節は？」 の 「1年」 (イチネン, 核 2 = 尾高想定) が自然に読まれるか。 アラビア数字 「1」 が 「イチ」 と読まれるか (「ワン」 等の英語読みにならないか)。 「1 + 年」 の連結が自然か |
| 「6つ」 (むっつ) 和語数詞 + 助数詞 | 「角 が 6つ ある 形」 の 「6つ」 (ムッツ, 核 1 想定: ム↓ッツ) が和語数詞として読まれるか。 アラビア数字 「6」 + 「つ」 で 「ロクツ」 「シックス つ」 等の誤読がないか — count_total で生成済の 「むっつ」 と同等の読みになっているか |
| 「一番」 (いちばん) | 「一番 暑い 季節」 「一番 近いから」 で 「一番」 (イチバン, 核 0 = 平板 想定) が副詞として自然に読まれるか。 「イチ + バン」 と切れていないか |
| 疑問詞: 「どんな / どこから / どうして / なぜ / いつ」 | 各疑問詞 (ド↑ンナ↑ 平板 / ド↑コカラ↑ 平板 / ド↑ーシテ↓ / ナ↓ゼ / イ↓ツ) が自然な疑問詞として読まれるか。 「どうして」 が長音含む 4 モーラとして自然に流れるか |
| 接続詞「なんという」 | 「〜を 何という？」 (ナ↑ントイウ↓ or ナ↑ンテイウ↓?) で 「何という」 が漢字 「何」 + 「という」 の連結として自然に読まれるか。 「何」 が 「ナン」 と読まれるか (「ナニ」 と誤読しないか) |
| 「朝」 (アサ) | 「涼しい 朝」 「朝、白い 靄が」 で 「朝」 (アサ, 核 1: ア↓サ) が自然に読まれるか。 「朝、」 の読点での息継ぎが不自然に長くないか |
| 「地方」 (チホー) | 「雪が たくさん 降る 地方を 何という？」 の 「地方」 (チホー, 核 1 想定: チ↓ホー) が自然に読まれるか。 「地方」 + 「を」 助詞の連結が自然か |
| 「電気」 (デンキ) | 「雲の 中で 電気が 溜まるから」 の 「電気」 (デ↓ンキ, 核 1) が 「天気」 (テンキ) と区別されて自然に読まれるか — どちらもテ系/デ系の似た発音なので濁音の区別が明瞭か |
| 長文の息継ぎ | 22 モーラ以上の長文 (「雷は 高い ものに よく 落ちる。どうして？」 = 24 モーラ、 「海や 川の 水が 空に 上っていくのを 何という？」 = 25 モーラ、 「息の 中の 水が 冷えて 粒に なるから」 = 21 モーラ) で途中の息継ぎが不自然に入っていないか。 句点 「。」 を含む文は自然な区切りになるか |
