📁 ステージ管理 (ブラウザ内)

    🔀 比較ペイン (上下分割)

    選ぶと下半分に保存済みステージを参考表示

    画像入りで保存 (1600px JPEG 圧縮)。複数バージョン保持してリスト切り替え可能。

    🌲 既存ステージから開く (サーバー)

    💾 保存場所について
    • 下書き (自動): ブラウザの localStorage に保存。 ページを閉じても復元される。
    • 名前付き保存: 「💾 保存」ボタンで localStorage に複数バージョン管理。 別ブラウザからは見えない。
    • JSON ダウンロード: 「⬇ ダウンロード」でブラウザの Downloads フォルダにファイル保存。 これを maze/imageStages/ に配置すれば本番に反映 (要 git commit + push)。
    • サーバー (本番): maze/imageStages/*.json に直接配置されたステージはこのページから再ロード可能。

    使い方

    1. 画像をドロップ
    2. 「スタート」「ゴール」モードで位置を 1 つずつ置く
    3. 「中継」モードで分岐や角に置く
    4. 「道を描く」モードで、ノード A をクリック → カーブの形を決めるアンカー点を順にクリック → ノード B をクリックして完成
    5. 「✋ 移動」モード: ノード・障害物・お邪魔虫をドラッグで動かす (新規配置なし)
    6. 「▶ 迷路で試す」 ボタンですぐ試遊 (デプロイ不要、sessionStorage 経由)
    7. OK なら「💾 JSON ダウンロード」→ maze/imageStages/<name>.json に配置 + 画像コピー → コミットで本番反映

    🕸️ クモの巣ステージ:

    1. 「🚧 障害物」モードで 🕸️ クモの巣 を配置
    2. 「🎒 アイテム」モード →「🕸️ クモの巣を動かす」を選択
    3. 黄色い破線リングをドラッグで位置調整、 サイズはスライダー
    4. (任意)「🧹 ほうき」で 拾うアイテムも配置

    カーブのコツ:
    ・道はクリックしたアンカー点を通る滑らかなカーブ (Catmull-Rom)
    ・直線: 中間点を打たず、ノード A → ノード B だけクリック
    ・強く曲げる: 曲がる位置にアンカーを増やす
    ・Esc / 右クリックで作成中の道を取消
    ・ノードに 30px 以内でクリックするとスナップ

    「迷路で試す」の仕組み:
    現在のノード/道/画像を sessionStorage にコピー → 新タブで maze/?image=__draft__ を開く。
    画像が小さい場合 (~5MB 以下) は dataURL も渡すので、画像が未デプロイでも遊べる。
    画像が大きい場合は、サーバーの maze/imageStages/<name>.<ext> を fetch (画像だけ事前にコミットしておけば OK)。
